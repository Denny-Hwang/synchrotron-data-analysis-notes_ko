# 참고 자료

이 섹션은 방사광 데이터 분석 노트를 위한 종합적인 참고 자료를 제공합니다.

## 내용

| 파일 | 설명 |
|------|------|
| [bibliography.bib](bibliography.bib) | 인용된 모든 논문의 BibTeX 파일 |
| [glossary.md](glossary.md) | 방사광 과학 용어 용어집 |
| [useful_links.md](useful_links.md) | URL: APS BER 프로그램, APS, 도구, 데이터셋, 튜토리얼 |

## 사용법

### BibTeX 인용

`bibliography.bib` 파일에는 이 아카이브 전체에서 참조된 모든 논문의 BibTeX 항목이
포함되어 있습니다. LaTeX에서 인용하려면:

```latex
\bibliography{08_references/bibliography}
\bibliographystyle{unsrt}
```

### 용어집

용어집은 이 아카이브 전체에서 사용되는 기술 용어를 정의합니다. 용어는 알파벳순으로
구성되어 있으며, 자세히 논의되는 섹션과 상호 참조됩니다.

### 링크

유용한 링크 파일은 아카이브에서 언급된 모든 외부 리소스에 대한 직접 URL을
카테고리별로 제공합니다.
