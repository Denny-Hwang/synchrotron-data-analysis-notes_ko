## 요약 (TL;DR)

본 논문은 빔라인의 검출기와 함께 배치된 엣지 GPU (NVIDIA Jetson)와 FPGA (Xilinx Alveo) 하드웨어에 딥러닝(deep learning) 기반 재구성을 배포함으로써, 최대 2~3 kHz 프레임 레이트에서의 실시간 스트리밍 타이코그래피(ptychography) 위상 복원(phase retrieval)을 시연한다. 본 시스템은 검출기 판독에서 재구성된 위상 이미지 디스플레이까지 5 ms 미만의 엔드투엔드 지연 시간을 달성하여, 원격 HPC로의 데이터 전송 병목을 제거하고 APS 타이코그래피 빔라인에서의 실시간 적응형 스캐닝(adaptive scanning)을 가능하게 한다. ZMQ 기반 스트리밍 통합이 재구성 파이프라인을 빔라인 제어 시스템에 연결하여 실시간 피드백을 제공한다.

---

## 배경 및 동기

타이코그래피는 집속 광학계의 한계를 넘는 해상도로 겹치는 결맞음 회절 패턴으로부터 시편의 진폭(amplitude)과 위상(phase) 모두를 복원한다. 기존의 반복 알고리즘(ePIE, difference map, RAAR)은 스캔 위치당 100~500회의 반복을 필요로 하여, 최신 검출기 프레임 레이트(1~10 kHz)에서 심각한 병목을 야기한다. APS-U는 결맞음 플럭스를 100~500배 증가시켜 데이터 속도를 10+ GB/s로 밀어올리며, 실시간 피드백을 필요로 하는 실험에서 이 격차는 지속 불가능해진다.

Guan 등(PtychoNet, 2019)의 선행 연구는 ~90% 속도 향상을 갖는 CNN 기반 타이코그래피 재구성을 시연했지만, 기존 GPU 서버에서의 배포는 상당한 데이터 전송 지연을 유지했다. 본 논문은 중요한 다음 단계를 밟아: 최적화된 신경망을 검출기와 함께 배치된 엣지 컴퓨트 하드웨어에 배포하고, RDMA 및 ZMQ 스트리밍으로 연결하여 진정한 실시간 운영을 달성한다.

## 방법

### 데이터

| 항목 | 세부사항 |
|------|---------|
| **데이터 출처** | 시뮬레이션 회절 패턴(훈련); APS 26-ID 및 2-ID-D의 실험 데이터(검증) |
| **시료 유형** | Siemens star 테스트 패턴, 집적 회로, 생물학적 세포 절편 |
| **데이터 차원** | 256x256 픽셀 회절 패턴; 128x128 픽셀 재구성 패치 |
| **전처리** | 강도의 로그 스케일링, 배경 차감, 패턴별 정규화 |

### 모델 / 알고리즘

1. **신경망 아키텍처**: 원래 PtychoNet (~800만 파라미터)보다 상당히 작은 약 200만 파라미터의 소형 인코더-디코더(encoder-decoder) CNN. 인코더는 잔차 연결(residual connections)과 스트라이드 합성곱(max-pooling 대신 strided convolutions)을 사용하는 4개의 합성곱 블록으로 다운샘플링을 수행한다. 디코더는 전치 합성곱(transposed convolutions)으로 인코더를 미러링한다. 소형 설계는 엣지 하드웨어 제약에 의해 주도됨: 메모리 사용량과 추론 지연 시간 모두 임베디드 장치의 자원 범위 내에 맞아야 한다.

2. **훈련 데이터**: 물리 기반 순방향 모델이 현실적인 포아송 노이즈(Poisson noise), 부분 결맞음 효과(partial coherence effects), 프로브 다양성을 포함한 시뮬레이션 회절 패턴을 생성한다. 약 200,000개의 패턴-시편 쌍이 생성된다. 그런 다음 시뮬레이션-실제(sim-to-real) 도메인 격차를 해소하기 위해 반복적으로 재구성된 실측 자료(ground truth)를 가진 소량의 실험 데이터(~5,000 패턴)를 사용한 전이 학습(transfer learning)이 적용된다.

3. **엣지 GPU 배포 (NVIDIA Jetson AGX Orin)**: 모델이 FP16으로 양자화되고 TensorRT로 컴파일되어, ARM 기반 GPU 아키텍처를 위한 레이어 퓨전 및 커널 자동 튜닝을 통해 회절 패턴당 ~0.5 ms를 달성한다.

4. **FPGA 배포 (Xilinx Alveo U250)**: INT8로 추가 양자화되고 Xilinx Vitis AI를 통해 파이프라인 데이터플로우 아키텍처로 합성되어, GPU 경로보다 상당히 낮은 전력 소비로 패턴당 ~0.3 ms를 달성한다.

5. **스트리밍 파이프라인**: Eiger 검출기로부터 RDMA를 통해 엣지 장치로 데이터가 흘러가며, 제어 워크스테이션을 우회한다. ZMQ PUB/SUB가 재구성된 위상 이미지를 시각화 대시보드와 Bluesky 제어 시스템에 스트리밍하여 적응형 스캔 결정을 지원한다.

6. **반복 정제 옵션**: CNN 출력은 지연 시간이 허용되는 경우 더 높은 충실도를 위해 5~10회의 ePIE 반복(200회 이상 대신)의 시드로 선택적으로 사용 가능하다.

### 파이프라인

```
Eiger 검출기 --> RDMA --> 엣지 장치 (Jetson/Alveo) --> CNN 추론
    --> ZMQ PUB --> [시각화 대시보드, Bluesky 스캔 컨트롤러]
    --> (선택 사항) GPU 서버에서 5-10 ePIE 정제 반복
```

이 아키텍처는 APS-U 데이터 속도를 위해 설계되었으며, 엣지 장치가 실시간 재구성을 처리하고 다운스트림 GPU 서버가 선택된 프레임의 선택적 정제에 사용 가능하다.

## 강점

- 엣지 하드웨어에서의 kHz 속도 타이코그래피 재구성을 시연 -- APS-U 데이터 속도에서의 적응형 스캐닝을 위한 핵심 마일스톤이자 전제 조건.
- 엣지 배포가 네트워크 병목을 제거: 데이터가 빔라인 허치를 벗어나지 않아 HPC 기반 워크플로우 대비 지연 시간이 수 자릿수 감소.
- FPGA가 서버 GPU 대비 5~7배 낮은 전력을 달성하며, 이는 지속 가능한 24/7 다중 빔라인 운영에 중요.
- 하이브리드 CNN + 소수 반복 ePIE가 다양한 실험 우선순위에 적합한 연속적으로 조절 가능한 품질/속도 트레이드오프를 제공.
- ZMQ 스트리밍이 주요 재설계 없이 기존 Bluesky/Ophyd 제어 인프라와의 깔끔한 통합을 제공.
- 시뮬레이션뿐 아니라 두 곳의 APS 빔라인에서 실험 데이터로 검증됨.

## 한계 및 격차

- 소형 CNN이 속도를 위해 재구성 품질을 교환: 달성 해상도가 동일 데이터에서 전체 반복 재구성으로 달성 가능한 ~10 nm 반피치 대비 ~15 nm 반피치로, 최고 해상도를 요구하는 실험에는 의미 있는 격차.
- 시뮬레이션에서 실험으로의 전이 학습에 빔라인별 보정 데이터가 필요; 다른 프로브 함수, 광자 에너지 또는 검출기 유형을 가진 빔라인 간 일반화는 시연되지 않음.
- FPGA INT8 양자화가 양자화 노이즈를 도입하며, 위상 이미지 정확도 및 공간 해상도에 대한 체계적 영향은 부분적으로만 특성화됨.
- 스트리밍 파이프라인이 2D 타이코그래피에 대해 시연됨; 여러 각도의 프로젝션을 축적하고 공동 처리해야 하는 3D 타이코그래피 토모그래피(ptycho-tomo)로의 확장은 논의되었으나 구현되지 않음.
- 전체 훈련 파이프라인 및 FPGA 합성 도구 체인이 오픈소스화되지 않아 FPGA 배포 경로에 대한 커뮤니티 재현성이 제한됨.
- 재구성된 위상에 대한 불확실성 정량화(uncertainty quantification) 없음: 사용자가 확신 있는 재구성 영역과 불확실한 영역을 구별할 수 없음.

## APS BER 프로그램과의 관련성

본 논문은 BER 프로그램의 실시간 피드백 아키텍처 및 엣지 컴퓨트 전략에 대한 핵심 참고 문헌이다:

- **지연 시간 벤치마크**: < 5 ms 엔드투엔드 지연 시간이 결맞음 이미징 빔라인에서의 BER 프로그램 실시간 의사결정 루프의 성능 목표를 설정하며, 밀리초 수준의 피드백이 기술적으로 달성 가능함을 시연한다.
- **엣지 컴퓨트 모델**: BER 프로그램은 밀리초 추론이 필요한 다른 측정 기법에 동일한 Jetson Orin / FPGA 엣지 배포 아키텍처를 채택할 수 있다: XRF 스펙트럼 피팅, XANES 분류, 회절 패턴 인덱싱.
- **적응형 타이코그래피**: 실시간 위상 이미지를 사용할 수 있으므로, BER 프로그램의 스캔 플래너가 오버랩 비율을 동적으로 조정하거나, 불필요한 위치를 건너뛰거나, 스캔 중 관심 특징으로 확대하여 빔타임당 정보 획득을 극대화할 수 있다.
- **APS-U 준비**: 시연된 2~3 kHz 처리량이 26-ID 및 2-ID-D에서 예상되는 APS-U 타이코그래피 프레임 레이트와 잘 일치하며, 이 접근법이 업그레이드된 광원 조건으로 확장됨을 확인한다.
- **ZMQ 통합**: ZMQ 기반 스트리밍 아키텍처가 BER 프로그램의 계획된 Bluesky/Tiled 데이터 인프라와 일치하여, 엣지 재구성 이미지를 실험 제어 루프에 원활하게 통합할 수 있다.
- **우선순위**: 핵심 -- BER 프로그램의 엣지-AI 타이코그래피 배포에 가장 직접적으로 관련된 연구.

## 실행 가능한 시사점

1. **엣지 하드웨어 조달**: APS-U 타이코그래피 엔드스테이션(26-ID, 2-ID-D)의 BER 프로그램 빔라인 허치에 랙 통합 및 냉각 솔루션과 함께 NVIDIA Jetson Orin 모듈과 Xilinx Alveo 보드를 확보.
2. **APS-U 프로브로 재훈련**: 초기 커미셔닝에서 측정된 프로브 함수, 결맞음 특성, 검출기 특성을 사용하여 APS-U 전용 훈련 데이터를 생성한 후, 전이 학습을 통해 사전 훈련된 모델을 미세 조정.
3. **ZMQ/Bluesky 통합**: 원활한 실시간 재구성, 시각화, 적응형 스캔 제어를 위해 검출기-엣지 간 RDMA 파이프라인을 BER 프로그램의 Bluesky/Tiled 데이터 인프라와 통합.
4. **타이코-토모 확장**: CNN 재구성 프로젝션을 축적하고 온라인 토모그래피 합성을 수행하여 스트리밍 모드에서 타이코그래피 토모그래피 체적을 재구성하는 3D 확장 개발.
5. **불확실성 모니터링**: CNN 출력 품질이 사용자 정의 임계값 이하로 저하될 때 전체 재구성으로의 폴백을 트리거하는 온라인 재구성 품질 추적(예: 가끔 수행되는 전체 반복 기준 프레임에 대한 실행 NRMSE) 구현.
6. **교차 모달 엣지 배포**: 본 연구를 다른 BER 프로그램 빔라인에서 XRF, XPCS, 토모그래피 재구성에 대한 엣지 추론 배포의 템플릿으로 활용.
