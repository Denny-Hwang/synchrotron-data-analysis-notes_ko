## 한 줄 요약

본 논문은 점수 기반 확산 모델(score-based diffusion model)을 희소 시점 및 제한 각도 프로젝션 데이터로부터의 CT 영상 재구성에 적용한다. 열화 영상에서 깨끗한 영상으로의 직접 매핑을 학습하는 GAN 기반 방법과 달리, 확산 모델은 깨끗한 영상 분포의 점수 함수(score function, 로그 확률 밀도의 기울기)를 학습하고 이를 사용해 역확산 과정(reverse diffusion process)을 통해 재구성을 반복적으로 정제한다. 이 접근법은 GAN 기반 방법 대비 우수한 SSIM 및 PSNR을 달성하고, 환각(hallucination) 아티팩트를 더 적게 생성하며, 재훈련 없이 임의의 샘플링 패턴에 유연하게 대응한다.

---

## 배경 및 동기

희소 시점 및 제한 각도 CT 재구성은 기존 해석적 방법(FBP)이 심각한 줄무늬 및 언더샘플링 아티팩트를 생성하는 부적절(ill-posed) 역문제이다. 딥러닝 접근법은 가능성을 보였지만 다음과 같은 한계가 있다:

- **GAN**: 미세한 구조를 환각할 수 있으며 훈련 불안정성에 민감.
- **지도 학습 CNN**: 과도하게 평활화하여 정량 분석에 중요한 미세한 세부 사항을 제거하는 경향.
- **물리 기반 반복 방법**: 계산 비용이 크고 신중한 정규화 조정이 필요.

점수 기반 확산 모델은 모드 붕괴를 피하고, 원리적인 확률론적 정식화를 제공하며, 샘플링 과정 중 물리 기반 데이터 일관성 제약을 통합할 수 있는 대안적 생성 프레임워크를 제공한다.

## 방법

### 데이터

| 항목 | 세부사항 |
|------|---------|
| **데이터 출처** | AAPM 저선량 CT 그랜드 챌린지; 임상 CT 데이터셋 |
| **시료 유형** | 의료 CT (복부, 흉부); 방사광 CT에 적용 가능 |
| **데이터 차원** | 512x512 재구성 슬라이스 |
| **전처리** | 희소 프로젝션으로부터의 FBP 초기 재구성; 정규화 |

### 모델 / 알고리즘

**점수 기반 확산 모델**: 노이즈 제거 점수 매칭(denoising score matching)을 사용하여 깨끗한 CT 영상 분포의 점수 함수를 추정하도록 신경망을 훈련한다. 재구성은 다음과 같이 진행된다:

1. 무작위 노이즈(또는 노이즈가 있는 FBP 초기화)에서 시작.
2. 학습된 역확산 과정을 통해 반복적 노이즈 제거.
3. 각 확산 단계에서, 측정된 사이노그램 데이터와 일관된 영상 집합으로 현재 추정치를 투영하여 데이터 일관성을 강제.

**데이터 일관성 투영**: 각 역확산 단계 후, 재구성은 순방향 모델(라돈 변환) 제약을 만족하도록 업데이트되어 측정된 프로젝션 데이터에 대한 충실도를 보장한다.

**GAN 대비 주요 장점**:

- 적대적 훈련 불안정성 또는 모드 붕괴 없음.
- 추론 시 임의의 샘플링 기하학(희소 시점, 제한 각도, 내부 토모그래피)에 재훈련 없이 유연하게 대응.
- 다중 샘플링 실행을 통해 자연스럽게 불확실성 추정치를 제공.

### 파이프라인

```
희소/제한 각도 사이노그램 데이터
  --> 초기 FBP 재구성 (선택 사항)
  --> 점수 기반 역확산 과정
      (반복적 노이즈 제거 + 데이터 일관성 투영)
  --> 고품질 재구성 영상
  --> (선택 사항) 불확실성 정량화를 위한 다중 샘플
```

## 주요 결과

| 지표                                | 값 / 결과                                                |
|-------------------------------------|-------------------------------------------------------|
| SSIM (60-시점 희소 CT)              | 0.96 (GAN 0.91, FBP 0.88 대비)                        |
| PSNR (60-시점 희소 CT)              | 38.2 dB (GAN 35.8, FBP 31.2 대비)                     |
| 환각 발생률                         | GAN 방법보다 거짓 구조가 현저히 적음                   |
| 유연성                              | 단일 모델이 여러 시점 수/기하학에 대해 작동             |
| 추론 시간                           | 슬라이스당 ~30초 (1000 확산 스텝, 단일 GPU)            |
| 불확실성 정량화                     | 샘플 간 분산이 실제 오류와 상관 관계                  |

- **Figure 3**: 60-시점 희소 CT로부터의 재구성 비교, 확산 모델의 GAN 대비 우수한 세부 사항 보존 및 더 적은 아티팩트를 보여줌.

## 강점

- **우수한 영상 품질**: 다중 희소 시점 설정에 걸쳐 SSIM 및 PSNR 지표에서 GAN 및 지도 학습 방법을 일관되게 능가.
- **환각 감소**: 데이터 일관성 제약을 가진 반복적 정제는 거짓 구조 생성 위험을 현저히 감소시키며, 이는 과학적 이미징에서 중요한 장점이다.
- **샘플링 유연성**: 단일 훈련 모델로 임의의 프로젝션 샘플링 패턴에서 재구성 가능하여, 서로 다른 수집 프로토콜에 대한 재훈련 필요성을 제거.
- **내장된 불확실성**: 다중 샘플링 실행이 아키텍처 수정 없이 자연스러운 불확실성 정량화를 제공.
- **원리적 프레임워크**: 잘 이해된 수렴 특성을 가진 점수 매칭 이론에 기반.

---

## 한계 및 격차

- **느린 추론**: 1000 확산 스텝으로 슬라이스당 ~30초는 순방향 방법(TomoGAN의 ~15 ms) 대비 수십~수백 배 느려 실시간 적용성이 제한됨.
- **계산 비용**: 훈련은 큰 GPU 메모리와 컴퓨팅 시간을 요구; 추론은 반복적 샘플링이 필요.
- **방사광 데이터에서 미검증**: 결과는 의료 CT에서 시연되었으며, 방사광 특유의 노이즈 및 아티팩트 특성에 대한 성능은 검증이 필요함.
- **가속-품질 트레이드오프**: 더 빠른 추론을 위해 확산 스텝을 줄이면 재구성 품질이 저하; 최적 스텝 수는 응용 의존적.
- **제한된 가용성**: 전체 코드 및 훈련된 모델이 공개적으로 이용 가능하지 않아, 커뮤니티의 채택과 벤치마킹을 저해.

## APS BER 프로그램과의 관련성

CT 재구성용 확산 모델은 BER 프로그램의 주요 목표를 다룬다:

- **생물학적 시료의 선량 감소**: 더 적은 프로젝션으로부터 고품질 영상을 재구성하여 직접적으로 방사선 선량을 감소시키며, 이는 생물학적 및 환경 시료에 중요.
- **유연한 수집 프로토콜**: 임의 샘플링 패턴 처리 능력은 APS-U에서 계획된 적응형 스캐닝 전략을 지원.
- **품질 관리를 위한 불확실성**: 내장된 불확실성 정량화는 신뢰 한계와 함께 신뢰할 수 있는 정량적 이미징에 대한 BER의 요구와 부합.
- **환각 안전성**: GAN 대비 감소된 환각 위험은 정량적 과학 분석에 대한 신뢰성을 높임.
- **우선순위**: **중상** -- 우수한 재구성 품질이지만, 추론 속도 한계가 실시간 파이프라인 배포 전 해결되어야 함. 가속화된 확산 샘플링 방법(DDIM, 일관성 모델)은 활발한 연구 분야로 이를 해결할 가능성이 있음.

## 실행 가능한 시사점

1. **방사광 CT 데이터에서 벤치마크**: APS 토모그래피 데이터에서 다양한 희소도 수준으로 점수 기반 확산 재구성을 평가.
2. **추론 가속화**: DDIM, 일관성 증류(consistency distillation), 또는 점진적 증류를 조사하여 샘플링 단계 수를 1000에서 10~50으로 감소.
3. **환각 발생률 비교**: 알려진 구조의 팬텀에서 TomoGAN 및 기타 방법 대비 환각 빈도를 체계적으로 비교.
4. **하이브리드 접근**: 초기 재구성에는 빠른 순방향 방법(TomoGAN), 고비중 정량 분석에는 확산 정제를 사용.
5. **불확실성 활용**: BER 데이터 파이프라인에서 자동 품질 표시를 위해 픽셀별 불확실성 맵 사용.

## 참고 및 논의

확산 모델은 역문제에 대한 생성 모델링의 현재 최전선을 대표한다. 계산 비용이 현재 실시간 배포를 제한하지만, 가속화된 샘플링의 빠른 발전은 차세대 재구성 파이프라인에 강력한 후보로 만든다. 본 논문의 환각 분석은 `review_hallucination_tomo_2021.md`의 체계적 연구를 보완하며, 확산 모델이 과학적 이미징에서 GAN보다 본질적으로 더 신뢰할 수 있다는 실질적 증거를 제공한다.
