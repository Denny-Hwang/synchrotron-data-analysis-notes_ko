## 요약 (TL;DR)

TomoGAN은 조건부 생성적 적대 신경망(conditional generative adversarial network)을 저선량 방사광 X선 토모그래피 재구성의 노이즈 제거에 적용하여, 방사선 노출 또는 수집 시간을 4~10배 줄이면서도 전량 스캔에 필적하는 이미지 품질을 달성한다. 적대적 훈련 전략 -- PatchGAN 판별기(discriminator)와 픽셀별 L1 및 VGG 지각적 손실(perceptual losses)의 결합 -- 은 기존 노이즈 제거 방법(BM3D) 또는 MSE 손실만으로 훈련된 CNN에 비해 우수한 지각적 품질과 에지 보존을 갖춘 결과를 생성한다.

---

## 배경 및 동기

APS 2-BM과 같은 빔라인에서의 방사광 X선 마이크로 토모그래피는 마이크론에서 서브마이크론 해상도의 3D 구조 이미징을 제공한다. 이 기법은 다양한 회전 각도에서 수백에서 수천 개의 프로젝션 이미지를 수집하여 3D 체적으로 계산적으로 재구성한다.

**선량-품질 트레이드오프**는 토모그래피에 근본적이다:

- **방사선 손상**: 생물학적 조직, 고분자, 수화된 재료, 배터리 전극은 모두 방사선에 민감하다. 높은 X선 선량은 이미징되는 바로 그 구조를 변화시켜, 과학적 결론을 훼손하는 아티팩트를 도입할 수 있다. 재료 진화를 추적하는 실시간(in-situ) 실험의 경우, 여러 시점에 걸친 누적 선량이 스캔당 선량 예산을 더욱 제약한다.
- **처리량**: 하루에 수십 개의 시료를 처리하는 고처리량 빔라인에서, 시료당 선량 감소는 더 짧은 스캔과 더 높은 시료 처리량을 가능하게 한다 -- 직접적인 용량 배수.
- **시간 해상도**: 동적 과정(균열 전파, 폼 붕괴, 배터리 사이클링)을 추적하는 시분해(4D) 토모그래피의 경우, 더 짧은 노출 시간은 개별 프레임이 더 노이즈가 많아지는 대가로 더 미세한 시간 샘플링을 가능하게 한다.

**기존 노이즈 제거 접근법**:

- 기존 공간 필터(중간값, 가우시안, 양방향)는 노이즈를 억제하지만 구조적 에지를 무차별적으로 흐리게 하여, 토모그래피가 드러내도록 설계된 미세한 세부 사항을 파괴한다.
- BM3D는 강력한 패치 기반 방법이지만 계산 비용이 높고 (~2D 슬라이스당 ~5초) 토모그래피 재구성의 특정 노이즈 특성(구조화된 줄무늬 아티팩트, i.i.d. 가우시안 노이즈만이 아닌)에 최적화되어 있지 않다.
- MSE(평균 제곱 오차)로 훈련된 CNN은 PSNR을 최대화하는 매끄러운 결과를 생성하지만, 미세한 텍스처, 입계, 기공 에지를 체계적으로 과도하게 평활화한다 -- 이러한 특징이 종종 토모그래피 분석의 주요 대상이다.
- GAN은 질적으로 다른 훈련 패러다임을 제공한다: 적대적 손실은 생성기가 실제 데이터 분포와 통계적으로 구별할 수 없는 이미지를 생성하도록 장려하여, MSE 훈련 네트워크가 폐기하는 텍스처와 고주파 콘텐츠를 보존한다.

## 방법

### 데이터

| 항목 | 세부사항 |
|------|---------|
| **데이터 출처** | APS 빔라인 2-BM; 쌍 저선량/고선량 수집 |
| **시료 유형** | 셰일 암석, 금속 폼, 마우스 뇌 조직 |
| **데이터 차원** | 2D 재구성 슬라이스, 2048x2048 픽셀; 훈련 패치 512x512 |
| **전처리** | FBP/gridrec (TomoPy)을 통한 토모그래피 재구성; 선량 감소(더 짧은 노출)와 포아송 노이즈 주입 모두로 생성된 저선량 데이터; 무작위 크롭, 뒤집기, 회전 증강 |

### 모델 / 알고리즘

**생성기(Generator)**: 스킵 연결을 가진 U-Net 스타일 인코더-디코더. 4개의 다운샘플링 블록(Conv2D -> BatchNorm -> LeakyReLU, 스트라이드-2 합성곱)이 특징 채널을 증가시키면서 공간 해상도를 축소. 병목부는 특징 추출을 위한 밀집 합성곱 블록(dense convolutional blocks) 사용. 해당 인코더 레이어로부터의 스킵 연결을 가진 4개의 업샘플링 블록이 전치 합성곱으로 해상도를 복원. tanh 활성화를 통한 단일 채널 출력.

**판별기(Discriminator)**: 단일 전체 이미지 예측이 아닌 각 70x70 픽셀 패치를 실제 또는 생성으로 분류하는 PatchGAN 아키텍처. 이는 지역 텍스처와 고주파 세부 사항의 보존을 장려한다. 아키텍처는 BatchNorm과 LeakyReLU를 가진 스트라이드 합성곱을 사용하여 실제/가짜 확률의 공간 맵을 생성.

**복합 손실 함수**:

```
L_total = lambda_adv * L_adversarial + lambda_pix * L_pixel + lambda_perc * L_perceptual
```

- L_adversarial: PatchGAN 판별기의 이진 교차 엔트로피
- L_pixel: 노이즈 제거 출력과 고선량 실측 자료 간의 L1 손실
- L_perceptual: VGG-16 특징 공간(conv3_3 레이어)에서의 특징 매칭 손실, 지각적 유사성 보존
- 전형적 가중치: lambda_adv = 0.01, lambda_pix = 1.0, lambda_perc = 0.1

**훈련**: Adam 옵티마이저, 학습률 2e-4, 배치 크기 16, ~200 에폭. ~10,000개의 쌍 슬라이스 패치에서 훈련. 하드웨어: 4x NVIDIA V100 GPU, ~8시간 훈련 벽시계 시간. 1:1 비율의 교대 생성기/판별기 업데이트.

### 파이프라인

```
고선량 프로젝션 데이터
  --> 선량 감소 (더 짧은 노출 또는 포아송 노이즈 주입)
  --> 토모그래피 재구성 (TomoPy를 통한 FBP/gridrec)
  --> 노이즈 재구성 슬라이스
  --> TomoGAN 생성기 (단일 순방향 패스)
  --> 노이즈 제거된 재구성 슬라이스
  --> (선택 사항) 분할 또는 정량적 분석
```

추론 시 각 2D 슬라이스가 독립적으로 처리된다. 전체 체적 노이즈 제거는 GPU에서 모든 슬라이스를 병렬 배치로 처리하여 달성된다.

## 강점

- **지각적 품질 보존**: 적대적 + 지각적 손실 조합이 순수 L1/L2 손실이 평활화하는 미세한 에지와 텍스처를 보존. 이는 단순한 미적 요소가 아님 -- 텍스처 보존은 기공률 분석, 입계 특성화, 형태학적 분할에 과학적으로 중요.
- **상당한 선량 감소**: 4~10배 선량 감소가 방사선 민감 시료, 고처리량 워크플로우, 시분해 이미징에 대한 주요 실질적 의미를 가짐.
- **빠른 추론**: ~15 ms/슬라이스가 실시간 재구성 파이프라인과 호환되어, 데이터 수집 중 즉석(on-the-fly) 노이즈 제거 가능.
- **실제 방사광 데이터**: 시뮬레이션만이 아닌 다양한 시료 유형에 걸쳐 APS 2-BM의 실제 빔라인 데이터에서 검증.
- **데이터와 함께 오픈소스**: 코드, 사전 훈련 모델, 훈련 데이터가 공개 이용 가능하여 높은 재현성 기준을 설정.
- **다중 손실 훈련 전략**: 세 가지 구성 요소 손실이 과도한 평활화(MSE만)와 훈련 불안정성(적대적만)의 극단을 피함.

## 한계 및 격차

- **환각 위험**: GAN은 그럴듯해 보이지만 물리적으로 잘못된 구조적 세부 사항을 생성할 수 있다. 과학적 이미징에서 이는 중대한 우려: 네트워크가 시료에 존재하지 않는 입계, 기공 구조, 개재물을 발명할 수 있다. 논문이 환각 빈도를 엄격하게 특성화하거나 감지 방법을 제공하지 않음.
- **모드 붕괴(mode collapse) 가능성**: GAN 훈련이 본질적으로 불안정하고 모드 붕괴에 취약. 다중 손실 구성이 이를 완화하지만 신중한 모니터링과 하이퍼파라미터 조정이 여전히 필요. 실패 사례 분석이 제공되지 않음.
- **슬라이스별 처리**: 각 2D 슬라이스가 독립적으로 노이즈 제거되며, 슬라이스 간 일관성 강제 없음. 3D 렌더링에서 깜빡임(flickering) 아티팩트와 z-방향 불일관성을 유발하여 3D 분할에 영향을 줄 수 있음.
- **쌍 훈련 데이터 요구**: 훈련에 일치하는 (노이즈, 깨끗한) 이미지 쌍이 필요하며, 이는 노이즈 시뮬레이션 또는 다중 선량 수준에서의 데이터 수집으로 생성해야 함 -- 둘 다 비용이 높고 실제 저선량 조건을 완벽하게 대표하지 않을 수 있음.
- **제한된 교차 시료 일반화**: 훈련에 대표되지 않는 형태학적으로 새로운 시료 유형에서 성능 저하. 각 새로운 재료 클래스에 재훈련 또는 미세 조정이 필요할 수 있음.
- **불확실성 정량화 없음**: TomoGAN이 픽셀별 신뢰도 추정 없이 단일 노이즈 제거 출력을 생성하여, 신뢰할 수 있게 복원된 특징과 그럴듯한 환각을 구별할 수 없음.

## APS BER 프로그램과의 관련성

TomoGAN은 BER 프로그램의 선량 최적화 및 처리량 목표에 직접적으로 관련된다:

- **적용 빔라인**: APS의 모든 토모그래피 빔라인, 2-BM, 32-ID, 마이크로-CT를 수행하는 모든 파트너 빔라인 포함. 타이코그래피 토모그래피 데이터에도 잠재적으로 적용 가능.
- **선량 적응형 스캐닝**: BER 프로그램의 스캔 플래너가 TomoGAN을 사용하여 방사선 민감 영역에서 선량을 줄이면서도 이미지 품질을 유지하여, 시료에 걸친 이질적 선량 할당을 가능하게 함.
- **실시간 파이프라인 통합**: ~15 ms 추론 시간이 APS-U 프레임 레이트에서의 스트리밍 재구성을 위한 BER 프로그램의 지연 시간 예산에 맞음. 재구성 후 콜백으로서 Bluesky/Tiled와의 통합이 간단함.
- **훈련 데이터 생성**: BER 프로그램이 커미셔닝 실행 중 쌍 선량 데이터셋의 수집을 자동화하여 시설 특화 TomoGAN 모델을 구축할 수 있음.
- **불확실성 확장**: TomoGAN을 Monte Carlo 드롭아웃 또는 앙상블 방법과 결합하면 프로그램의 자동 품질 관리를 위한 픽셀별 불확실성 맵을 제공할 수 있음.
- **우선순위**: **높음** -- 노이즈 제거는 보편적 수요이며, TomoGAN은 Argonne 계보를 가진 잘 검증된 접근법을 대표. 배포는 환각 감지 프로토콜과 함께 이루어져야 함.

---

## 실행 가능한 시사점

1. **APS-U 토모그래피에 배포**: APS-U 2-BM 커미셔닝 데이터에서 TomoGAN을 재훈련하고 업데이트된 검출기 노이즈 프로파일에 대해 벤치마킹.
2. **불확실성 추가**: 노이즈 제거 출력과 함께 신뢰도 맵을 생성하는 MC-드롭아웃 또는 심층 앙상블 변형을 구현하여, 잠재적으로 환각된 특징을 표시.
3. **3D 확장**: 슬라이스 간 일관성을 강제하기 위해 아키텍처를 2.5D(다중 슬라이스 입력) 또는 전체 3D 합성곱 블록으로 확장.
4. **자기 지도 학습 대안**: 쌍 고선량 실측 자료의 필요성을 제거하는 Noise2Noise 또는 Noise2Void 훈련 전략을 탐색하여, 새로운 시료 유형에 대한 재훈련 비용 절감.
5. **Bluesky와 통합**: APS BER 빔라인에서 실시간 재구성 중 즉석 노이즈 제거를 위해 TomoGAN 추론을 Bluesky 콜백으로 패키징.
6. **환각 벤치마킹**: 다양한 선량 감소 수준에서의 환각 발생률을 정량화하기 위해 알려진 구조의 팬텀을 사용하는 테스트 스위트 개발.

## 참고 및 논의

TomoGAN은 방사광 토모그래피 커뮤니티에서 가장 많이 인용된 딥러닝 논문 중 하나이며, AI 기반 노이즈 제거의 벤치마크로 널리 채택되었다. Argonne 계보(실제 2-BM 데이터로 APS에서 개발)가 APS BER 프로그램에 특히 관련되게 만든다. `review_fullstack_dl_tomo_2023.md`에서 리뷰된 풀스택 파이프라인 논문이 완전한 토모그래피 워크플로우 내에서 TomoGAN 스타일 노이즈 제거가 맞는 위치에 대한 더 넓은 맥락을 제공하며, `review_realtime_uct_hpc_2020.md`의 실시간 HPC 파이프라인은 TomoGAN이 스트리밍 분석 시스템에 통합된 방법을 시연한다.
