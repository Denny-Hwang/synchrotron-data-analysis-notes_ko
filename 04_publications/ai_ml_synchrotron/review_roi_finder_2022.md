## 요약 (TL;DR)

ROI-Finder는 PCA 기반 차원 축소(dimensionality reduction)와 퍼지 k-평균 클러스터링(fuzzy k-means clustering) 및 의사결정 지원 점수 함수(scoring function)를 결합하여 방사광 XRF 매핑의 관심 영역(region-of-interest, ROI) 선택을 자동화한다. 이 파이프라인은 넓은 간격 탐사에서 미세 스캔까지의 전환 시간을 수십 분의 수동 검사에서 60초 미만의 자동 분석으로 단축하며, 전문가 선택 ROI와 약 85%의 일치도를 달성한다. 이 접근법은 완전히 비지도 학습(unsupervised)이며, 레이블된 훈련 데이터가 필요 없고, GPU 하드웨어 없이 표준 빔라인 워크스테이션에서 실행된다.

---

## 배경 및 동기

3세대 및 4세대 방사광에서의 고해상도 X선 형광(XRF) 현미경은 서브-100 nm 해상도로 원소 분포를 매핑할 수 있지만, 전체 시료를 최고 해상도로 스캔하는 것은 시간적으로 불가능하다. APS 2-ID-E와 같은 XRF 마이크로프로브 빔라인에서의 표준 워크플로우는 두 단계 계층을 따른다:

1. **넓은 간격 탐사 스캔**: 전체 시료 또는 넓은 관심 영역에 걸쳐 원소 분포를 매핑하는 저해상도, 넓은 시야 래스터 스캔. 스텝 크기는 보통 1-5 마이크로미터이며 짧은 체류 시간.

2. **미세 목표 스캔**: 탐사 결과의 시각적 검사를 기반으로, 실험자가 고해상도 스캔(50-300 nm 스텝 크기, 픽셀당 더 긴 체류 시간)을 위한 하나 이상의 하위 영역을 수동으로 선택.

수동 ROI 선택 단계는 진정한 운영 병목이다:

- **주관성**: 다른 실험자가 동일한 탐사 맵에서 다른 ROI를 선택. 다원소 XRF 데이터셋에서 "흥미로운" 영역을 구성하는 것에 대한 표준화되고 재현 가능한 기준이 없음.
- **시간 비용**: 복잡하고 이질적인 시편(생물학적 조직, 환경 입자, 지질학적 절편)의 경우, 10개 이상의 원소 채널과 그 공간 상관의 수동 검사에 탐사 스캔당 10-30분 소요.
- **확장성 장벽**: 자율 및 고처리량 실험 워크플로우는 인간 개입 없이 탐사 스캔과 목표 측정 간의 루프를 닫기 위해 자동화된 ROI 선택이 필요.
- **선행 기술 한계**: 이전 접근법은 가장 과학적으로 흥미로운 영역을 종종 정의하는 다원소 상관을 무시하고, 단순한 단일 원소 임계화(thresholding) 또는 수동 등고선 그리기에 의존.

ROI-Finder는 탐사 스캔의 전체 다원소 정보 내용을 고려하는 객관적이고 재현 가능하며 신속한 추천 시스템을 제공하기 위해 개발되었다.

### 모델 / 알고리즘

ROI-Finder는 전적으로 고전 ML에 구축된 세 단계 비지도 파이프라인이다:

**1단계 -- PCA 차원 축소**:

다원소 XRF 맵 스택(픽셀 x 원소)이 2D 행렬로 재구성되어 주성분 분석(Principal Component Analysis)에 적용된다. 누적 분산의 >95%를 포착하는 첫 k개의 주성분(보통 k = 3-5)이 유지된다. PCA는 화학적으로 구별되는 영역을 구분하는 지배적 공간 패턴을 보존하면서 원소 채널을 비상관화하고 차원을 축소하는 역할을 한다. PCA 로딩(loadings) 자체가 해석 가능: PC1은 종종 총 원소 함량(세포 대 배경), PC2는 세포 내 대비(핵 대 세포질), 그리고 높은 PC는 미량 원소 핫스팟에 대응한다.

**2단계 -- 퍼지 K-평균 클러스터링**:

PCA 축소된 픽셀 표현이 퍼지 c-평균(fuzzy c-means, FCM)으로 클러스터링된다. 이는 각 픽셀이 경성 이진 할당이 아닌 각 클러스터에 대해 연속적 소속도(membership degree, 0에서 1)를 받는 연성 클러스터링(soft clustering) 알고리즘이다. 클러스터 수 c는 퍼지 분할 계수(fuzzy partition coefficient, FPC)를 통해 선택되며, 세포 XRF 데이터에 대해 보통 c = 3-8이다. 퍼지 소속도 맵은 경성 분할보다 풍부한 공간 특성을 제공하며, 경계 영역과 혼합 화학 조성의 픽셀이 여러 클러스터에 대한 부분적 소속을 유지한다.

**3단계 -- ROI 점수 매기기 및 추천**:

퍼지 소속도 맵이 임계화되어 이진 후보 마스크를 생성한다. 연결 성분 분석(connected component analysis)이 이산 ROI 후보를 식별한다. 복합 점수 함수가 구성 가능한 기준에 따라 후보의 순위를 매긴다: 공간 범위(고해상도 스캔에 충분한 크기), 화학적 이질성(ROI 내 PCA 분산으로 측정되는 흥미로운 다원소 시그니처의 존재), 형태학적 컴팩트니스(분절된 영역보다 연속적 영역 선호). 운영자는 다양한 실험 목표에 맞게 이러한 기준의 상대적 가중치를 조정할 수 있다. 상위 순위 ROI가 빔라인 스캔 소프트웨어에 직접 입력 가능한 경계 상자(bounding boxes)로 반환된다.

### 파이프라인

```
다원소 XRF 탐사 맵
  --> 스펙트럼 피팅 (MAPS)
  --> 픽셀별 원소 농도 맵
  --> PCA 차원 축소 (k = 3-5 성분)
  --> 퍼지 c-평균 클러스터링 (c = 3-8 클러스터, FPC 선택)
  --> 소속도 맵 임계화 + 연결 성분 분석
  --> ROI 점수 매기기 (이질성, 크기, 컴팩트니스)
  --> 순위화된 ROI 경계 상자 --> 빔라인 스캔 컨트롤러
```

구현은 NumPy, scikit-learn (PCA), scikit-fuzzy (FCM)을 사용한 Python 기반이다. Matplotlib으로 구축된 대화형 GUI가 운영자에게 클러스터 맵 시각화, 파라미터 조정, 추천 승인/거부를 허용한다.

## 강점

- **완전 비지도**: 레이블된 훈련 데이터가 필요 없어 주석 노력 없이 모든 XRF 빔라인에 즉시 배포 가능. 파이프라인이 각 탐사 스캔에 독립적으로 적응.
- **경량 및 고속**: PCA + 퍼지 k-평균이 GPU 요구 사항을 피하여 빔라인 제어 워크스테이션에서 실시간 실행 가능. < 60초 처리 시간이 탐사와 미세 스캔 사이의 유휴 시간에 맞음.
- **예시 데이터와 함께 오픈소스**: 독립적 재현, 커뮤니티 확장, 다른 빔라인 데이터 포맷 적응을 용이하게 함.
- **구성 가능한 점수 함수**: 운영자가 포괄적 커버리지에서 집중 조사까지 다양한 실험 우선순위에 맞게 화학적 다양성 대 공간적 컴팩트니스 대 영역 크기의 가중치를 조정할 수 있음.
- **전 세계 모든 XRF 마이크로프로브 빔라인에서 상당한 빔타임을 낭비하는 진정한 운영 병목을 직접 해결.**
- **해석 가능한 중간 출력**: PCA 로딩과 퍼지 소속도 맵이 물리적으로 의미 있으며, ROI 좌표 이상의 추가적 과학적 통찰을 제공.

## 한계 및 격차

- **PCA 선형성**: PCA는 원소 채널 간 선형 상관만 포착. 비선형 관계(예: 복잡한 화학양론적 제약, 매트릭스 흡수 효과)는 PCA에 보이지 않음. UMAP, t-SNE, 오토인코더(autoencoders)와 같은 비선형 대안이 더 풍부한 표현을 제공할 수 있음.
- **이진 최종 출력**: 내부적으로 퍼지 클러스터링을 사용하지만, 최종 ROI 추천은 경성 임계화 단계에 의존하여 확률적 소속 정보를 폐기. 완전히 확률적인 ROI 순위가 더 정보적일 것임.
- **k와 c 선택 필요**: PCA 성분 수와 클러스터 수를 지정하거나 경험적으로 선택해야 함. FPC가 가이드를 제공하지만, 다양한 시료 유형에 걸쳐 검증된 완전 자동화되고 강건한 선택 방법은 없음.
- **대안 ML 방법과의 비교 없음**: 동일 데이터셋에서 GMM 클러스터링, HDBSCAN, 스펙트럼 클러스터링, 딥러닝 기반 분할에 대한 벤치마킹이 없음.
- **평가에서 제한된 시료 다양성**: 주로 생물학적 세포 맵과 지질학적 절편에서 시연; 배터리 재료, 촉매, 환경 입자에 대한 적용 가능성은 명시적으로 검증되지 않음.
- **비공식적 정확도 평가**: 85% 전문가 일치도가 엄격한 평가자 간 신뢰도 통계, 맹검 평가(blinded evaluation), 통계적 유의성 검정 없이 보고됨.
- **APS 데이터 포맷에 대한 GUI 결합**: 시각화 대시보드가 APS 특화 HDF5 스키마를 가정; 다른 시설 포맷으로의 적응에는 수동 구성이 필요.

## APS BER 프로그램과의 관련성

ROI-Finder는 지능형 빔라인 자동화라는 BER 프로그램의 목표에 핵심적인 스캔 전략 최적화 문제를 직접 해결한다:

- **적용 빔라인**: APS의 모든 XRF 마이크로프로브 빔라인, 업그레이드된 섹터 2 빔라인(2-ID-E, 2-ID-D) 및 Bionanoprobe (9-ID) 포함. 탐사 후 목표 워크플로우를 사용하는 마이크로-XRD 빔라인에도 적용 가능.
- **자율 스캐닝 기준선**: ROI-Finder의 순위화된 후보가 BER 프로그램의 적응형 샘플링 알고리즘(베이지안 최적화, 강화 학습 기반 스캔 플래너)의 사전 분포(prior)를 초기화할 수 있음.
- **Bluesky 통합 템플릿**: 파이프라인이 고전 ML 분석을 Bluesky/Tiled 데이터 인프라에 통합하는 모델을 제공. 넓은 간격 탐사 -> ROI-Finder -> 미세 스캔을 실행하는 Bluesky 계획이 자율 실험 루프를 닫을 수 있음.
- **낮은 배포 장벽**: 비지도, GPU 불필요, 오픈소스, 고속. 1-2주 내에 파트너 빔라인에 배포 가능.
- **우선순위**: **높음** -- BER 프로그램의 핵심 미션에 직접 부합하는 실제 운영 수요를 해결하는 실용적이고 배포 가능한 도구.

## 실행 가능한 시사점

1. **벤치마크 기준선**: BER 프로그램의 XRF 빔라인 스캔 계획 평가 스위트에서 ROI-Finder를 "고전 ML 기준선"으로 통합.
2. **자동 k 선택**: APS BER 빔라인에 배포하기 전에 실루엣(silhouette) 기반 또는 갭 통계(gap-statistic) 자동 클러스터 수 선택으로 퍼지 k-평균 확장.
3. **포맷 어댑터**: 핵심 알고리즘을 수정하지 않고 APS-U 및 NSLS-II XRF 데이터 포맷을 ROI-Finder 파이프라인에 수집하는 얇은 HDF5 어댑터 작성.
4. **하이브리드 접근법**: ROI-Finder 클러스터 맵을 BER 프로그램의 베이지안 최적화 루프에 적응형 스캐닝을 위한 정보적 공간 사전 분포로 공급.
5. **지연 시간 목표**: < 60초 실행 시간이 유용한 기준점을 설정; BER 프로그램의 DL 기반 대안은 동등하거나 더 빠른 추론을 목표로 해야 함.
6. **전문가 피드백 수집**: 빔타임 중 전문가 선택과 함께 ROI-Finder를 실행하고, 향후 학습 기반 점수 매기기를 위한 수락/거부 데이터 수집.

## 참고 및 논의

ROI-Finder는 XRF 실험 루프에 ML을 도입하는 실용적이고 저위험 진입점을 대표한다. 퍼지 c-평균 구성 요소는 `review_xrf_gmm_2013.md`에서 리뷰된 GMM 접근법과 관련이 있다; 둘 다 연성/확률적 공간 분할을 제공하지만, FCM은 가우시안 분포 특징을 가정하지 않으며 계산적으로 더 효율적이다. 자연스러운 다음 단계는 ROI-Finder를 심층 잔차 초해상도 접근법(`review_deep_residual_xrf_2023.md`)과 결합하여, 먼저 넓은 간격 탐사 해상도를 계산적으로 향상한 후 향상된 맵에서 ROI를 식별하는 것이다.
