## 요약 (TL;DR)

본 연구는 딥러닝(deep learning) 기반 재구성을 검출기와 함께 배치된 엣지 GPU 및 FPGA 하드웨어에 배포함으로써, 원격 HPC 클러스터로의 데이터 전송 지연 병목을 제거하고 최대 2 kHz 프레임 레이트에서 실시간 스트리밍 타이코그래피(ptychography) 위상 복원(phase retrieval)을 시연한다.

---

## 배경 및 동기

타이코그래피(ptychography)는 렌즈 한계를 넘는 해상도로 시편의 진폭(amplitude)과 위상(phase) 모두를 복원하는 결맞음 회절 이미징(coherent diffraction imaging) 기법이다. 기존의 반복 알고리즘(ePIE, DM, RAAR)은 스캔 위치당 수백 번의 반복이 필요하여 현대 검출기 프레임 레이트(kHz)에서의 실시간 재구성을 불가능하게 만든다. 한편, APS-U는 결맞음 플럭스를 100~500배 증가시켜 10+ GB/s의 데이터 속도를 생성할 예정이다. 데이터 수집 속도와 재구성 처리량 간의 격차는 적응형 스캐닝(adaptive scanning)(실시간 재구성 이미지를 기반으로 오버랩, 체류 시간 또는 스캔 경로를 조정)을 가능하게 하는 실시간 피드백을 방해한다. 본 논문은 빔라인에 직접 배치된 엣지 컴퓨트 하드웨어(GPU용 NVIDIA Jetson, FPGA용 Xilinx Alveo)에 신경망 기반 위상 복원을 배포하여 이 격차를 해결한다.

---

## 방법

1. **신경망 아키텍처**: 단일 회절 패턴(diffraction pattern)을 입력으로 받아 복소값 시편 투과 함수(complex-valued object transmission function)를 출력하는 소형 인코더-디코더(encoder-decoder) CNN (~200만 파라미터). 아키텍처는 잔차 블록(residual blocks)을 사용하며, 반복 재구성 연산자를 근사하도록 훈련된다.
2. **훈련 데이터**: 현실적인 노이즈, 프로브 다양성(probe diversity), 부분 결맞음(partial coherence)을 포함한 물리 기반 순방향 모델로부터 시뮬레이션된 회절 패턴을 생성. 소량의 실험 데이터를 사용한 전이 학습(transfer learning)을 도메인 적응(domain adaptation)에 적용.
3. **엣지 GPU 배포**: 훈련된 모델을 양자화(quantize, FP16)하고 NVIDIA Jetson AGX Orin용 TensorRT로 컴파일. 회절 패턴당 ~0.5 ms의 추론(inference) 달성.
4. **FPGA 배포**: 모델을 고정 소수점(INT8) 표현으로 변환하고 Vitis AI를 사용하여 Xilinx Alveo U250 FPGA에 합성. 더 낮은 전력 소비로 패턴당 ~0.3 ms 추론 달성.
5. **스트리밍 파이프라인**: 검출기(Eiger)로부터 RDMA를 통해 엣지 장치로 데이터가 직접 전달되며, 제어 워크스테이션을 우회함. 재구성된 위상 이미지가 5 ms 미만의 총 지연 시간으로 시각화 대시보드에 스트리밍됨.
6. **반복 정제(iterative refinement)**: CNN 출력은 시간이 허용되는 경우 더 높은 충실도를 위해 소수의 ePIE 반복(200회 이상 대신 5~10회)의 시드로 선택적으로 사용 가능.

## 강점

- 적응형 스캐닝의 전제 조건인 kHz 속도의 실시간 타이코그래피 재구성을 APS-U 데이터 속도에서 시연.
- 엣지 배포가 네트워크 병목을 제거: 초기 재구성을 위해 데이터가 빔라인 허치(hutch)를 벗어나지 않으므로 지연 시간이 극적으로 감소.
- FPGA 구현이 GPU 서버 대비 6~7배 낮은 전력으로 경쟁력 있는 처리량을 달성하며, 이는 지속 가능한 시설 운영에 중요.
- 하이브리드 CNN + 소수 반복 ePIE 접근법이 다양한 실험 우선순위에 적합한 조절 가능한 품질/속도 트레이드오프를 제공.
- 시뮬레이션뿐 아니라 실험적 타이코그래피 데이터에서 검증됨.

---

## 한계 및 격차

- 소형 CNN이 속도를 위해 재구성 품질을 교환; 최종 해상도는 동일 데이터에서 전체 반복 재구성으로 달성 가능한 ~10 nm 대비 ~15 nm.
- 시뮬레이션에서 실험으로의 전이 학습에 빔라인별 보정 데이터가 필요; 빔라인 간 일반화는 시연되지 않음.
- FPGA INT8 양자화가 양자화 노이즈를 도입; 위상 이미지에서의 양자화 유발 아티팩트에 대한 체계적 특성화가 제한적.
- 스트리밍 파이프라인이 2D 타이코그래피에 대해 시연됨; 3D 타이코그래피 토모그래피(ptycho-tomo)로의 확장은 논의되었으나 구현되지 않음.
- 전체 훈련 파이프라인 및 FPGA 합성 도구 체인이 오픈소스화되지 않아 FPGA 배포에 대한 커뮤니티 재현성이 제한됨.

## APS BER 프로그램과의 관련성

본 논문은 BER 프로그램의 실시간 피드백 아키텍처에 대한 핵심 참고 문헌이다:

- **지연 시간 요구 사항**: < 5 ms 엔드투엔드 지연 시간이 결맞음 이미징 빔라인에서의 BER 프로그램 실시간 의사결정 루프의 벤치마크를 설정한다.
- **엣지 컴퓨트 전략**: BER 프로그램은 밀리초 수준의 추론이 필요한 다른 측정 기법(XRF 스펙트럼 피팅, XANES 분류)에 동일한 Jetson Orin / FPGA 엣지 배포 모델을 채택할 수 있다.
- **적응형 타이코그래피**: 실시간 위상 이미지를 사용할 수 있으므로, BER 프로그램의 스캔 플래너가 스캔 중 오버랩 비율을 조정하거나, 불필요한 위치를 건너뛰거나, 관심 특징으로 확대할 수 있다.
- **APS-U 준비**: 2~3 kHz 처리량이 26-ID 및 2-ID-D에서 예상되는 APS-U 타이코그래피 프레임 레이트와 잘 일치한다.

## 실행 가능한 시사점

1. **엣지 하드웨어 조달**: APS-U 타이코그래피 엔드스테이션의 BER 프로그램 빔라인 허치에 NVIDIA Jetson Orin 모듈과 Xilinx Alveo 보드를 확보.
2. **APS-U 프로브로 재훈련**: 커미셔닝에서 측정된 프로브 함수와 결맞음 특성을 사용하여 APS-U 전용 훈련 데이터를 생성.
3. **스트리밍 통합**: 검출기-엣지 간 RDMA 파이프라인을 BER 프로그램의 Bluesky/Tiled 데이터 인프라와 통합하여 원활한 실시간 재구성.
4. **타이코-토모 확장**: CNN 재구성 프로젝션을 축적하여 스트리밍 모드에서 타이코그래피 토모그래피 체적을 재구성하는 3D 확장 개발.
5. **품질 모니터링**: CNN 출력 품질이 사용자 정의 임계값 이하로 저하될 때 전체 반복 재구성으로의 폴백을 트리거하는 온라인 NRMSE 추적 구현.
