## 방법

### 데이터

| 항목 | 세부사항 |
|------|----------|
| **데이터 출처** | _빔라인, 시뮬레이션, 공개 데이터셋 등_ |
| **시료 유형** | _생물학적 세포, 합금, 고분자 등_ |
| **데이터 차원** | _예: 256x256 픽셀, 2048 프로젝션, N 에너지 채널_ |
| **전처리** | _정규화, 필터링, 정렬 등_ |

### 모델 / 알고리즘

_기술적 접근법을 상세히 기술하세요:_

- 아키텍처 (예: U-Net, ResNet, GAN, GMM, PCA + 클러스터링)
- 손실 함수 및 훈련 전략
- 주요 하이퍼파라미터
- 하드웨어 요구사항 (GPU, 메모리, 훈련 시간)

### 파이프라인

_원시 데이터에서 최종 출력까지의 전체 워크플로우를 기술하세요:_

```
Raw data --> Preprocessing --> Model --> Post-processing --> Output
```

_스트리밍, 엣지 컴퓨팅, HPC 등 주요 통합 지점을 포함하세요._
