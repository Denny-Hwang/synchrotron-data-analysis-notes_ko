TomocuPy

GPU-accelerated tomographic reconstruction using CuPy and CUDA for real-time processing at APS beamlines.

tomography GPU reconstruction CUDA CuPy
Used at: 2-BM 7-BM 32-ID

Architecture

TomocuPy is a GPU-accelerated tomographic reconstruction package built on CuPy. It replaces CPU-bound NumPy operations with GPU kernels, achieving 100–300× speedup on NVIDIA A100 GPUs.

Key architectural components:

  • Preprocessing pipeline — flat-field correction, ring removal, phase retrieval
  • Reconstruction engine — filtered back-projection (FBP) and gridrec algorithms
  • Memory manager — chunk-based processing for datasets exceeding GPU memory
  • CLI interface — drop-in replacement for TomoPy CLI

Pros / Cons

Advantages

  • 100–300× faster than TomoPy for FBP reconstruction
  • Drop-in replacement: same CLI and data format
  • Supports Data Exchange HDF5 format natively

Trade-offs

  • Requires NVIDIA GPU with CUDA 11.2+
  • Fewer reconstruction algorithms than TomoPy
  • CuPy dependency adds installation complexity

Code Example

Python import tomocupy # Load a Data Exchange HDF5 file data = tomocupy.read_hdf5("/data/sample.h5") # Reconstruct with FBP on GPU recon = tomocupy.recon( data, algorithm="fbp", center=1024.5, gpu_id=0, )
