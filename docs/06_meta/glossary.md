# Glossary

| Term | Definition |
|------|-----------|
| **ADR** | Architecture Decision Record — a document capturing a significant design decision, its context, and consequences. |
| **APS** | Advanced Photon Source — a synchrotron light source at Argonne National Laboratory providing high-brightness X-ray beams for scientific research. |
| **APS-U** | APS Upgrade — the comprehensive upgrade completed in 2024 that delivers 500× brighter X-rays through a multi-bend achromat lattice. |
| **Beamline** | A dedicated experimental station at a synchrotron facility where X-ray beams are directed and conditioned for specific measurement techniques. |
| **BER** | Biological and Environmental Research — a DOE Office of Science program funding research in biological systems, climate, and environmental science. |
| **Cluster** | In the eBERlight Explorer IA, one of three task-oriented groups (Discover, Explore, Build) that organize note content by user intent. |
| **DOE** | U.S. Department of Energy. |
| **eBERlight** | The integrated BER research program at APS combining multiple X-ray techniques for biological and environmental science. |
| **EPICS** | Experimental Physics and Industrial Control System — a set of software tools for building distributed control systems at scientific facilities. |
| **Frontmatter** | YAML metadata block at the top of a markdown file, delimited by `---`, containing structured fields like title, tags, and modality. |
| **HDF5** | Hierarchical Data Format version 5 — a file format for storing large scientific datasets with a tree-like structure of groups and datasets. |
| **Modality** | A specific X-ray measurement technique (e.g., tomography, XRF microscopy, ptychography, spectroscopy, crystallography, scattering). |
| **PRD** | Product Requirements Document — specifies the functional and non-functional requirements for a product. |
| **Ptychography** | A coherent diffraction imaging technique that achieves sub-10nm spatial resolution by scanning a focused X-ray beam and computationally reconstructing the phase. |
| **SemVer** | Semantic Versioning — a versioning scheme using MAJOR.MINOR.PATCH to communicate the nature of changes. |
| **XAS** | X-ray Absorption Spectroscopy — a technique measuring the energy-dependent absorption of X-rays to determine local atomic structure and chemistry. |
| **XANES** | X-ray Absorption Near-Edge Structure — the portion of an XAS spectrum near the absorption edge, sensitive to oxidation state and coordination. |
| **XRF** | X-ray Fluorescence — the emission of characteristic X-rays from a material excited by an incident X-ray beam, used for elemental mapping. |
| **Zoom level** | In the eBERlight Explorer 4-zoom model, a level of specificity: L0 (Map), L1 (Section), L2 (Topic), L3 (Source). |
