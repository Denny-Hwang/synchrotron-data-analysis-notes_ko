# 방사광 데이터를 위한 노이즈 제거 방법론(Denoising Methods)

## 개요

노이즈 제거(Denoising)는 방사광 과학에서 매우 중요합니다. 그 이유는:
1. 방사선 선량 감소(생물학적 시료 보호)가 노이즈를 유발함
2. 더 빠른 스캔(시간 분해 연구를 위한)은 픽셀당 광자 수가 적어짐을 의미
3. APS-U 이후 데이터 속도가 저선량, 고처리량 수집을 요구

## 방법론 분류

| 범주 | 학습 데이터 | 예시 |
|----------|--------------|---------|
| **지도 학습(Supervised)** | 쌍을 이루는 노이즈/깨끗한 이미지 | TomoGAN, 표준 노이즈 제거 CNN |
| **자기 지도 학습(Self-supervised)** | 노이즈 쌍 (깨끗한 목표 없음) | Noise2Noise, Noise2Void |
| **비지도 학습(Unsupervised)** | 쌍이 필요 없음 | Deep image prior, BM3D |
| **GAN 기반** | 쌍 데이터 + 적대적 학습 | TomoGAN (지도 학습 + GAN 결합) |
