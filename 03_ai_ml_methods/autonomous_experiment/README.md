# 자율 실험 조종(Autonomous Experiment Steering)

## 개요

자율 실험 방법은 AI/ML을 활용하여 방사광 실험 중 실시간 의사 결정을 수행합니다 — 측정 영역 선택, 매개변수 최적화, 그리고 수집되는 데이터에 기반한 데이터 수집 전략 적응을 포함합니다.

## 왜 자율 실험인가?

APS-U 이후 자율 실험을 촉진하는 과제들:
1. **데이터 볼륨**: 사람이 TB 규모의 데이터를 실시간으로 검토할 수 없음
2. **빔 타임 부족**: 빔 타임(Beam Time) 시간당 과학적 산출물 극대화 필요
3. **복잡성**: 다중 매개변수 최적화가 인간의 처리 능력을 초과
4. **재현성**: 알고리즘적 의사 결정은 재현 가능하며 문서화됨

## 자율 수준

| 수준 | 설명 | 인간의 역할 | 예시 |
|-------|-------------|-----------|---------|
| **0** | 수동 | 모든 의사 결정 | 전통적 실험 |
| **1** | 자문 | 최종 결정 | ROI-Finder가 세포를 제안 |
| **2** | 감독하의 자율 | 재정의 가능 | 승인을 받는 자동 정렬 |
| **3** | 조건부 자율 | 범위 설정 | 지정 영역 내 자율 스캐닝 |
| **4** | 완전 자율 | 모니터링만 | 자율 주행 빔라인 |

현재 상태: 대부분의 시스템이 수준 1-2에 해당하며, 연구는 수준 3-4를 향해 진행 중입니다.

## 방법론 범주

| 방법 | 의사 결정 유형 | 입력 | 동작 |
|--------|--------------|-------|--------|
| **ROI 선택** | 어디를 측정할 것인가 | 저해상도 데이터 | 영역 우선순위 설정 |
| **매개변수 최적화** | 어떻게 측정할 것인가 | 품질 지표 | 노출, 에너지 조정 |
| **적응적 스캐닝(Adaptive Scanning)** | 어떤 해상도로 | 진행 중인 결과 | 스텝 크기 변경 |
| **이상 탐지(Anomaly Detection)** | 언제 플래그할 것인가 | 데이터 스트림 | 운영자에게 경고 |
| **종료 기준(Stopping Criteria)** | 언제 중단할 것인가 | 누적 데이터 | 수집 종료 |
