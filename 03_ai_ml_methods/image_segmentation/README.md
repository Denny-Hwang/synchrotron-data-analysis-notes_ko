# 방사광 데이터를 위한 이미지 분할(Image Segmentation)

## 개요

이미지 분할(Image Segmentation)은 방사광 이미지를 의미 있는 영역(예: 세포, 기공, 입자, 상(Phase))으로 구획합니다. 모든 하위 정량 분석이 정확한 분할에 의존하므로, 이는 가장 중요한 분석 단계 중 하나입니다.

## 분할 유형

| 유형 | 출력 | 적용 예시 |
|------|--------|-------------------|
| **의미론적(Semantic)** | 픽셀 단위 클래스 레이블 | µCT에서 상(Phase) 식별 (기공/입자/광물) |
| **인스턴스(Instance)** | 개별 객체 경계 | XRF 맵에서 세포 탐지 |
| **파놉틱(Panoptic)** | 의미론적 + 인스턴스 결합 | 완전한 조직/시료 분석 |

## 방법론 비교

| 방법 | 유형 | 학습 데이터 | 속도 | 품질 | 적합한 용도 |
|--------|------|--------------|-------|---------|----------|
| **Otsu 임계값(Thresholding)** | 전통적 | 없음 | ★★★★★ | ★★ | 고대비 이진 분할 |
| **워터셰드(Watershed)** | 전통적 | 없음 | ★★★★ | ★★★ | 접촉하는 객체 |
| **랜덤 워커(Random Walker)** | 전통적 | 시드 | ★★★ | ★★★★ | 저대비 |
| **U-Net** | DL 의미론적 | 50-500 레이블 | ★★★★ | ★★★★★ | 범용 |
| **nnU-Net** | DL 의미론적 | 50-500 레이블 | ★★★ | ★★★★★ | 자동 구성 |
| **Cellpose** | DL 인스턴스 | 사전 학습됨 | ★★★★ | ★★★★★ | 세포 형태 객체 |
| **StarDist** | DL 인스턴스 | 사전 학습됨 | ★★★★ | ★★★★ | 볼록 객체 |
| **Mask R-CNN** | DL 인스턴스 | 100+ 레이블 | ★★★ | ★★★★ | 복잡한 형태 |

## 방사광 데이터의 주요 과제

1. **볼륨 크기**: 2048³ 복셀 = 80억 복셀; GPU 메모리에 들어갈 수 없음
2. **레이블 부족**: 방사광 데이터의 전문가 주석이 매우 비용이 높음
3. **도메인 시프트(Domain Shift)**: 한 빔라인/시료에서 학습된 모델이 일반화되지 않을 수 있음
4. **클래스 불균형**: 관심 특징이 종종 볼륨의 극히 일부분
5. **3D 대 2D**: 진정한 3D 분할 대 슬라이스별 처리 후 일관성 유지
