# MAPS -- XRF 스펙트럼 분석 소프트웨어

## 개요

MAPS는 Advanced Photon Source에서 정량적 X선 형광 (XRF) 마이크로프로브 데이터 분석을 위한 표준 소프트웨어 패키지입니다. Stefan Vogt와 동료들이 개발한 이 소프트웨어는 스캐닝 XRF 빔라인 (2-ID-D, 2-ID-E, Bionanoprobe)에서 수집된 에너지 분산 스펙트럼을 처리하여 정량적 원소 농도 맵을 생성합니다.

## 주요 기능

- **픽셀별 스펙트럼 피팅** -- 각 XRF 스펙트럼을 원소 방출선과 배경의 조합에 피팅하여 정량적 농도 (ug/cm2)를 산출합니다.
- **배치 ROI 적분** -- 전체 피팅이 필요하지 않을 때 관심 영역 (ROI) 윈도우를 통한 빠른 원소 정량화.
- **표준 시료 교정** -- 알려진 참조 표준 (예: NBS 박막)을 피팅하여 검출기 감도 곡선을 도출합니다.
- **다중 검출기 병합** -- 적절한 입체각 가중치를 적용하여 여러 형광 검출기의 데이터를 결합합니다.
- **출력 형식** -- MAPS 스키마를 따르는 HDF5 파일로 피팅된 맵을 기록합니다 (`/MAPS/XRF_fits`, `/MAPS/channel_names`, `/MAPS/scalers`).

## 일반적인 워크플로우

1. 빔라인에서 XRF 스캔 및 참조 표준 스캔을 수집합니다.
2. 원시 MDA 또는 HDF5 파일을 MAPS에 로드합니다.
3. 표준을 사용하여 검출기 에너지 축과 감도를 교정합니다.
4. 픽셀별 피팅을 실행합니다 (매트릭스 보정을 포함한 가우시안 피크 형상).
5. 피팅된 원소 맵을 HDF5로 내보내어 후속 분석에 사용합니다.

## 소프트웨어 접근

- MAPS는 Linux 및 macOS용 컴파일된 바이너리로 배포됩니다.
- 빔라인 컴퓨팅 환경을 통해 APS 사용자에게 제공됩니다.
- 문의: Stefan Vogt, XSD / APS.

## APS와의 통합

ROI Finder 파이프라인 (`../roi_finder/` 참조)은 MAPS HDF5 출력을 기본 입력 형식으로 사용합니다. `/MAPS/XRF_fits` 데이터셋은 세그멘테이션, 특징 추출 및 클러스터링에 사용되는 다중 원소 농도 맵을 제공합니다.
