# 결정학을 위한 AI/ML 방법

## 개요

AI/ML은 결정 검출부터 구조 검증까지 거대분자 결정학 파이프라인의 모든 단계를
변혁하고 있습니다. 가장 영향력 있는 발전은 AlphaFold의 구조 예측 혁명이지만,
실험 데이터 처리에서도 상당한 진전이 이루어지고 있습니다.

## ML 문제 유형

| 문제 | 유형 | 입력 | 출력 |
|------|------|------|------|
| 결정 센터링 | 객체 검출 | 광학 이미지 | 결정 좌표 |
| 인덱싱 | 분류 + 회귀 | 회절 패턴 | 공간군 + 셀 |
| 해상도 추정 | 회귀 | 회절 이미지 | 해상도 (Å) |
| 위상 복원 | 생성/회귀 | 구조인자 | 전자밀도 |
| 모델 구축 | 구조 예측 | 서열 + 밀도 | 원자 좌표 |
| 리간드 동정 | 분류 | 전자밀도 블롭 | 리간드 정체 |

## 방법

### 1. AlphaFold 통합

**AlphaFold** (DeepMind, 2021)은 분자 치환 위상 결정을 위한 정확한 초기 모델을
제공함으로써 결정학을 근본적으로 변화시켰습니다.

**워크플로우**:
```
단백질 서열 → AlphaFold 예측 → 초기 모델
                                    │
실험 데이터 (MTZ) ─────────────→ 분자 치환 (PHASER)
                                    │
                               위상 결정된 전자밀도
                                    │
                               정밀화 → 최종 구조
```

**영향**:
- 이전에 풀 수 없었던 구조(MR을 위한 상동체 없음)가 이제 위상 결정 가능
- 데이터에서 구조까지의 시간을 일/주에서 시간 단위로 단축
- 새로운 PDB 등록의 ~85%가 현재 AlphaFold 모델 사용

**한계**:
- AlphaFold 모델이 잘못된 국소 구조를 가질 수 있음
- 다량체 복합체와 유연한 영역은 여전히 어려움
- 모델 편향: 신중하게 검증하지 않으면 정밀화된 구조에 AlphaFold 오류가 남을 수 있음

### 2. 자동 결정 센터링

빔라인의 광학 현미경 이미지로부터의 ML 기반 결정 검출.

**접근법**:
- **CNN 기반 검출**: 루프 영역을 결정/비결정으로 분류
- **분할**: 결정 경계 검출을 위한 U-Net
- **회절 기반**: X선 래스터 스캔 + CNN으로 최적 회절 스팟 식별

**아키텍처**: 일반적으로 ResNet 또는 EfficientNet 백본 + 검출 헤드
**학습 데이터**: 빔라인 카메라의 수천 개 주석 달린 결정 이미지
**속도**: 시료당 1초 미만 (실시간 호환)

### 3. 자동 인덱싱 및 처리

**DIALS (첨단 광원을 위한 회절 적분)**:
- 적응적 임계값을 이용한 스팟 찾기
- 도전적 사례(다중 격자, 약한 회절)를 위한 ML 향상 인덱싱
- 공간군 결정을 위한 베이지안 접근법

**xia2**: 최적 파라미터를 선택하는 자동화된 처리 파이프라인
- 처리 전략 선택을 위한 결정 트리 로직
- 방사선 손상 및 아이스 링에 대한 이상 탐지

### 4. 직렬 결정학 데이터 처리

SSX는 많은 수의 부분 관측으로 인해 고유한 ML 과제를 제시합니다:

- **히트 찾기**: CNN이 회절 이미지를 결정/공백으로 분류 (10-90%의 공백 필터)
- **인덱싱률 최적화**: ML이 이미지별 최적 인덱싱 파라미터를 예측
- **병합**: 최적 강도 병합을 위한 기대값 최대화 알고리즘
- **동형분류**: 다형성을 처리하기 위해 단위 셀별 결정 클러스터링

### 5. 전자밀도 해석

- **자동 모델 구축**: ARP/wARP, Buccaneer가 통계적 방법으로 사슬 추적
- **리간드 피팅**: ML이 전자밀도 블롭을 알려진 리간드 라이브러리로 분류
- **물 배치**: 정렬된 물 분자 식별을 위한 CNN 기반 접근법
- **검증**: ML 향상 이상치 검출이 포함된 MolProbity

## 현재 한계 및 기회

### 한계
- 분자 치환에서의 AlphaFold 모델 편향은 신중한 처리 필요
- 빔라인 특화 ML(결정 센터링, 히트 찾기)을 위한 제한된 레이블 데이터
- SSX 데이터 처리는 실시간 분석에 여전히 계산 비용이 높음
- 유연한 영역과 본질적 무질서 단백질은 여전히 도전적

### 기회
- **실시간 처리**: 빔타임 중 ML 안내 데이터 수집 결정
- **미세결정 검출**: 딥러닝으로 10 µm 미만 결정의 향상된 검출
- **공동 정밀화**: 정밀화에서 실험 데이터와 ML 예측 결합
- **동적 구조**: 회절 데이터에서 구조적 앙상블을 추출하는 ML 방법
- **De novo 위상 결정**: 회절 강도에서 직접 위상 예측 (활발한 연구 분야)
