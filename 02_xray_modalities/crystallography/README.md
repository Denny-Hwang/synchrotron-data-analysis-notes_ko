# 결정학 (MX / SSX)

## 원리

**거대분자 결정학(Macromolecular Crystallography, MX)**은 X선이 결정 격자와 상호작용할 때
생성되는 회절 패턴을 분석하여 생물학적 거대분자(단백질, 핵산, 복합체)의 3D 원자 구조를
결정합니다.

### 물리적 기초
- X선은 결정 내 원자의 전자구름에서 산란됨
- 특정 각도에서의 보강 간섭이 회절 패턴을 생성 (브래그 법칙: nλ = 2d sin θ)
- 각 회절 스팟 강도는 구조 정보를 인코딩
- 위상 문제: 강도는 측정되지만 위상은 계산적으로 결정해야 함

### APS에서의 변형법

| 변형법 | 설명 | 결정 크기 | 데이터 수집 |
|--------|------|----------|------------|
| **MX** | 표준 단결정 회전 | 20–500 µm | 단일 결정, 180° 회전 |
| **SSX** | 직렬 싱크로트론 결정학 | 1–20 µm | 수천 개의 무작위 결정 |
| **마이크로결정학** | 소형/도전적 결정 | 5–50 µm | 집속 마이크로빔 |
| **MAD/SAD** | 이상분산 위상 결정 | 임의 | 다파장 |

## 실험 장치

```
X선 소스 (언듈레이터)
    │
    ▼
단색화기 (에너지/파장 선택)
    │
    ▼
집속 광학계 (KB 미러)
    │
    ▼
고니오미터 + 결정 ──→ 회절 패턴 ──→ 픽셀 어레이 검출기
    │                                     (EIGER/PILATUS)
    ▼
극저온 스트림 (100 K)
```

### APS BER 빔라인
- **21-ID-D**: 가변 에너지 (6.5–20 keV), EIGER 16M, 자동 시료 교환기
- **21-ID-F**: 고정 에너지 (12.7 keV), 고처리량 스크리닝
- **21-ID-G**: 가변, 직렬 결정학 역량

## 데이터 수집 프로세스

1. **결정 장착**: 루프/메쉬 마운트, 100 K로 급속 냉각 (또는 SSX의 경우 상온)
2. **센터링**: X선 빔에 결정 정렬 (자동 또는 ML 보조)
3. **전략 결정**: 테스트 이미지 수집, 최적 회전 범위 및 노출 결정
4. **데이터 수집**: 회절 프레임을 기록하며 각도 범위를 통해 결정 회전
5. **데이터 처리**: 인덱싱 → 적분 → 스케일링 → 병합 → 위상 결정 → 모델 구축 → 정밀화

### 일반적 파라미터
- **에너지**: 12.66 keV (SAD용 Se 흡수단) 또는 12.0 keV (표준)
- **회전 범위**: 180° (MX) 또는 프레임당 0.1–0.5°
- **노출**: 프레임당 0.01–1초
- **총 프레임**: 데이터셋당 360–3600
- **총 시간**: 데이터셋당 5–30분

## APS BER 응용

- **구조 효소학**: 토양 내 탄소/질소 순환과 관련된 효소 구조
- **금속단백질 구조**: 철-황 클러스터, 망간 중심, 아연 핑거
- **막단백질**: 양분 수송체, 유출 펌프
- **시간분해**: 상온에서 SSX로 포착된 효소 중간체
