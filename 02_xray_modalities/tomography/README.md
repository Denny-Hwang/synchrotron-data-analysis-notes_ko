# 토모그래피 (µCT / 나노-CT)

## 원리

**X선 컴퓨터 단층촬영(CT)**은 서로 다른 회전 각도에서 획득한 일련의 2D 투영 이미지로부터
시료의 3D 내부 구조를 재구성합니다. 이 기법은 비파괴적이며 시료를 자연 상태(습윤, 동결
또는 In-situ)에서 이미징할 수 있습니다.

### 물리적 기초
- X선이 시료를 통과하면서 감쇠됨 (Beer-Lambert 법칙)
- 감쇠는 물질 밀도, 조성, X선 에너지에 의존
- 많은 각도(일반적으로 180° 또는 360°)에서 투영을 수집하면 3D 감쇠 계수 분포를
  수학적으로 재구성할 수 있음 (라돈 변환 / 역변환)

### 대비 메커니즘

| 메커니즘 | 기초 | 최적 대상 | APS-U 영향 |
|----------|------|----------|------------|
| **흡수** | 질량 감쇠 | 고밀도/고원자번호 물질 | 표준 |
| **위상대비** | 굴절률 | 연조직, 저원자번호 | 획기적 향상 |
| **엣지 향상** | 자유공간 전파 | 계면, 경계 | 향상 |
| **격자 기반** | 탈봇 간섭계 | 정량적 위상 | 새로운 역량 |

### APS에서의 변형법

| 변형법 | 해상도 | 시야 | 스캔 시간 | 응용 |
|--------|--------|------|----------|------|
| **µCT** | 0.5–5 µm | 1–10 mm | 1–60분 | 토양 응집체, 뿌리, 암석 |
| **나노-CT (TXM)** | 50–100 nm | 50–200 µm | 30–120분 | 세포 초미세구조 |

| **4D CT** | 1–5 µm | 1–5 mm | 시계열 | In-situ 반응 |

## 실험 장치

```
X선 소스 (편향자석 또는 언듈레이터)
    │
    ▼
단색화기 또는 핑크빔 (속도를 위해)
    │
    ▼
회전 스테이지 위 시료 ──→ 0°에서 180° 회전
    │
    ▼
신틸레이터 (X선을 가시광으로 변환)
    │
    ▼
광학 배율 (대물렌즈)
    │
    ▼
카메라 (sCMOS: PCO Edge 5.5, Grasshopper, Oryx)
```

### APS BER 빔라인
- **2-BM-A**: 주력 µCT, 고속 토모그래피, 환경 셀, 10–40 keV
- **7-BM-B**: 넓은 에너지 범위 (6–80 keV), 이중 에너지 역량
- **32-ID-B/C**: 투과 X선 현미경 (TXM), 존 플레이트 광학, 나노-CT

## 데이터 수집 프로세스

1. **시료 장착**: 회전 스테이지의 튜브/캐필러리/환경 셀에 시료 배치
2. **플랫필드 수집**: 시료 없는 이미지 (I₀) 정규화용
3. **다크필드 수집**: 빔 차단 시 이미지 (검출기 배경)
4. **투영 수집**: 시료 회전, 각 각도에서 이미지 수집
5. **재구성**: 투영으로부터 3D 볼륨 계산

### 일반적 파라미터

| 파라미터 | µCT (2-BM) | 나노-CT (32-ID) |
|----------|-----------|----------------|

| **투영** | 900–3600 | 720–1500 |
| **이미지 크기** | 2048×2048 또는 4096×4096 | 1024×1024~2048×2048 |
| **노출/프레임** | 1–100 ms | 0.5–5 s |
| **총 스캔 시간** | 2–60분 | 30–120분 |
| **픽셀 크기** | 0.65–6.5 µm | 50–100 nm |

## APS BER 응용

- **토양과학**: 3D 공극 네트워크 분석, 응집체 구조, 뿌리-토양 계면
- **식물과학**: 토양 내 뿌리 시스템 구조, 관다발 구조
- **지구화학**: 암석 미세구조, 유체 포유물 분석, 광물 텍스처
- **환경**: 바이오필름 구조, 다공성 매질 내 오염물질 분포
- **In-situ 연구**: 건습 순환, 광물 용해/침전
