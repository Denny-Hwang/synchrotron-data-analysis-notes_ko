# 토모그래피 재구성 방법

## 개요

토모그래피 재구성은 2D 투영 이미지 세트로부터 3D 볼륨을 계산하는 수학적 과정입니다.
이것은 X선 토모그래피의 핵심 계산적 과제이며, 재구성 알고리즘의 선택은 이미지 품질,
속도, 선량 요구사항에 큰 영향을 미칩니다.

## 수학적 기초

순방향 모델은 객체 함수 f(x,y)를 측정된 투영 p(θ,t)에 연결합니다:

```
p(θ, t) = ∫ f(x, y) δ(x cos θ + y sin θ - t) dx dy    (라돈 변환)
```

재구성은 p(θ,t)로부터 f(x,y)를 복원합니다 — **역 라돈 변환**.

## 해석적 방법

### 필터 역투영(FBP)

싱크로트론 토모그래피의 표준 주력 알고리즘.

**알고리즘**:
1. 각 사이노그램 행에 램프 필터 적용 (푸리에 도메인)
2. 필터링된 사이노그램을 재구성 그리드에 역투영
3. 모든 각도의 기여를 합산

**수학**:
```
f(x,y) = ∫₀^π p̃(θ, x cos θ + y sin θ) dθ

여기서 p̃는 필터링된 투영: p̃ = F⁻¹{|ω| · F{p}}
```

**특성**:
- 빠름: 슬라이스당 O(N² log N)
- 조밀하게 샘플링된 각도 필요 (나이퀴스트: N_angles ≈ π/2 × N_pixels)
- 노이즈에 민감 (램프 필터가 고주파 노이즈를 증폭)
- 일반적 필터: Ram-Lak (램프), Shepp-Logan, Hann, Butterworth

**구현**: TomoPy `recon(algorithm='gridrec')`, TomocuPy

### Gridrec

푸리에 공간에서 격자 보간을 사용하는 최적화된 FBP 변형:
- 직접 FBP보다 5-10배 빠름
- TomoPy의 기본 알고리즘
- 표준 FBP와 거의 동일한 결과

## 반복적 방법

### SIRT (동시 반복 재구성 기법)

**알고리즘**:
```
f^(k+1) = f^(k) + λ · Aᵀ · C · (p - A · f^(k))

여기서:
  A = 순방향 투영 연산자
  Aᵀ = 역투영 연산자
  C = 대각 보정 행렬
  λ = 이완 파라미터
```

**특성**:
- FBP보다 우수한 노이즈 처리
- 사전 지식 통합 가능 (양수성, 매끄러움)
- FBP보다 10-100배 느림
- 최소 제곱 해로 수렴

### MLEM (최대 우도 기대값 최대화)

**알고리즘**:
```
f^(k+1) = f^(k) · (Aᵀ · p / (A · f^(k))) / (Aᵀ · 1)
```

**특성**:
- 곱셈 업데이트 → 자연스럽게 양수성 강제
- 통계적으로 동기 부여 (포아송 노이즈 모델)
- 느린 수렴, 정규화 필요 (OS-MLEM 변형이 더 빠름)

### CGLS (켤레 기울기 최소 제곱)

- 켤레 기울기 방법으로 ||Af - p||² 최소화
- 빠른 수렴 (일반적으로 5-20 반복 충분)
- 내장 정규화 없음

## 딥러닝 방법

### 신경망 재구성

| 방법 | 유형 | 핵심 혁신 |
|------|------|----------|
| **FBPConvNet** | 후처리 | CNN이 FBP 재구성을 노이즈 제거 |
| **LEARN** | 학습된 반복 | 학습된 파라미터를 가진 언롤된 최적화 |
| **iCT-Net** | 직접 역변환 | 엔드투엔드 사이노그램 → 볼륨 |
| **INR** | 암시적 신경 | 좌표 MLP가 연속 볼륨을 표현 |

### 암시적 신경 표현(INR)

신경망으로 파라미터화된 연속 함수로 볼륨을 표현:

```
f_θ(x, y, z) → 감쇠 값

여기서 θ는 측정된 투영에 맞추어 최적화되는 네트워크 파라미터
```

**장점**:
- 연속 표현 (임의 해상도)
- 희소/불규칙 각도 샘플링의 자연스러운 처리
- 4D 재구성을 위한 시간 차원 통합 가능: f_θ(x, y, z, t)

**과제**:
- 볼륨당 학습 시간 (분에서 시간)
- 현재 중간 해상도(~512³)로 제한
- 신중한 아키텍처 설계 필요 (위치 인코딩, SIREN)

## APS에서의 재구성 파이프라인

```
사이노그램 (전처리 완료)
    │
    ├─→ 회전 중심 찾기 (자동: Vo 알고리즘)
    │
    ├─→ 위상 복원 (선택사항: Paganin 방법)
    │
    ├─→ 재구성
    │       ├── 실시간: TomocuPy (GPU, 초)
    │       ├── 표준: TomoPy gridrec (CPU, 분)
    │       └── 고품질: 반복법 (CPU/GPU, 시간)
    │
    ├─→ 후처리
    │       ├── 링 아티팩트 제거
    │       ├── 노이즈 제거 (DNN 또는 가우시안/중앙값)
    │       └── 강도 정규화
    │
    └─→ 재구성된 3D 볼륨
            └── 분할 및 분석 준비 완료
```
