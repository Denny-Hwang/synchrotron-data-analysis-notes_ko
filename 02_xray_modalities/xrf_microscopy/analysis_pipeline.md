# XRF 현미경 분석 파이프라인

## 개요

XRF 분석 파이프라인은 원시 검출기 스펙트럼을 정량적 원소 맵으로, 궁극적으로는
공간적 원소 분포에 대한 과학적 통찰로 변환합니다.

## 파이프라인 단계

```
원시 검출기 데이터 (픽셀별 스펙트럼)
    │
    ├─→ 1. 전처리 (데드타임, 정규화)
    │
    ├─→ 2. 스펙트럼 피팅 (MAPS, PyXRF)
    │       출력: 원소 강도 맵
    │
    ├─→ 3. 정량화 (기본 파라미터)
    │       출력: 농도 맵 (µg/cm²)
    │
    ├─→ 4. 공간 분석
    │       ├── 분할 (세포, 입자, 상)
    │       ├── 상관 분석 (원소 공동위치화)
    │       └── 통계 분석 (PCA, 클러스터링)
    │
    └─→ 5. 해석
            ├── 원소 연관 식별
            ├── 화학종 추론
            └── 생물학적/지질학적 과정 해석
```

## MAPS 소프트웨어

**MAPS**(미시분석 툴킷)는 X선 과학부에서 개발한 APS의 주요 XRF 분석 소프트웨어입니다.

### 기능
- 기본 파라미터를 사용한 전체 스펙트럼 피팅
- 다중 피팅 방법: ROI, 가우시안, SVD, NNLS
- 대규모 스캔 데이터셋의 배치 처리
- 픽셀별 및 평균 스펙트럼 분석
- 자기흡수 효과에 대한 흡수 보정
- HDF5, TIFF, ASCII로 내보내기

### 스펙트럼 피팅 방법

#### ROI (관심 영역)
```
intensity = Σ (counts[ch_low : ch_high])
```
- 가장 간단한 방법: 예상 피크 위치 주변 에너지 창의 카운트 합산
- 빠르지만 피크 겹침과 배경 변동에 민감
- 잘 분리된 강한 피크에 가장 적합

#### 가우시안 피크 피팅
```
I(E) = Σ_elements [ A_i × G(E; E_i, σ) ] + background(E)

여기서 G(E; E_i, σ) = (1/σ√2π) × exp(-(E-E_i)²/2σ²)
```
- 각 피크를 가우시안 프로파일로 피팅
- 중간 수준의 피크 겹침 처리
- 피크 면적, 위치, 폭 제공

#### 전체 스펙트럼 피팅 (NNLS)
```
측정 스펙트럼 = Σ_i (c_i × 참조_스펙트럼_i) + 잔차

||측정 - Σ c_i × ref_i||² 최소화, c_i ≥ 0 조건
```
- 참조 스펙트럼 사용 (기본 파라미터에서 계산 또는 측정된 표준)
- 피크 겹침, 이스케이프 피크, 산란에 대한 최상의 처리
- 가장 정확한 정량화

### MAPS 정량화

기본 파라미터 접근법:
```
C_element = I_measured / (I_theoretical × T × Ω × ε)

여기서:
  I_theoretical = 단위 농도당 이론적 수율
  T = 시료 투과 보정
  Ω = 검출기 입체각
  ε = 형광 에너지에서의 검출기 효율
```

## 공간 분석 방법

### 원소 공동위치화

원소 쌍 간의 공간적 상관관계를 정량화:

```python
# 원소 맵 간 피어슨 상관관계
from scipy.stats import pearsonr

r_Fe_P, p_value = pearsonr(fe_map.flatten(), p_map.flatten())
# r > 0.5: 공동위치화, r ≈ 0: 독립, r < -0.5: 반상관
```

### RGB 합성 맵

시각화를 위해 세 원소를 R, G, B 채널에 할당:
- **생물학적**: R=Fe, G=Zn, B=P (세포 연구에 일반적)
- **환경적**: R=Fe, G=Ca, B=S (토양/광물 연구용)
- 공동위치화는 색상 혼합으로 나타남 (예: Fe+Zn = 노란색)

### 다중 원소 맵에 대한 PCA

주성분 분석으로 다중 원소 데이터의 차원을 축소:

```python
from sklearn.decomposition import PCA

# 모든 원소 맵 스택: shape (Npixels, Nelements)
X = np.column_stack([maps[e].flatten() for e in elements])
X_standardized = (X - X.mean(axis=0)) / (X.std(axis=0) + 1e-10)

pca = PCA(n_components=5)
scores = pca.fit_transform(X_standardized)
loadings = pca.components_

# 점수를 이미지 차원으로 재형성
pc1_map = scores[:, 0].reshape(ny, nx)
```

PCA 식별 결과:
- **PC1**: 일반적으로 전체 신호 강도 (밝기)
- **PC2+**: 원소 연관 패턴 (예: Fe-P 대 Ca-S 그룹화)
- 로딩은 각 성분에 기여하는 원소를 나타냄

### 클러스터링

원소 조성에 따라 픽셀을 그룹화:

- **K-means**: 경성 클러스터링, 각 픽셀을 하나의 클러스터에 할당
- **퍼지 k-means**: 연성 클러스터링, 클러스터별 멤버십 확률 (ROI-Finder에서 사용)
- **GMM**: 가우시안 혼합 모델, 확률적 클러스터 할당
- **HDBSCAN**: 밀도 기반, 임의 형태의 클러스터 식별

## 고급 분석

### µ-XANES 이미징

XRF + XANES 결합: 각 픽셀에서 전체 XANES 스펙트럼 수집
- 공간적으로 화학종을 매핑
- 각 위치에서 에너지 스캐닝 필요 (매우 느림)
- 대안: 비율 매핑을 위해 2-3개 핵심 에너지 선택

### 토모그래피 XRF

3D 원소 매핑:
1. 여러 회전 각도에서 XRF 맵 수집
2. 필터 역투영을 사용하여 3D 원소 분포 재구성
3. 과제: 3D에서의 자기흡수 보정

### 상관 이미징

XRF와 결합:
- **타이코그래피**: 구조 + 조성 (같은 빔라인: 2-ID-E)
- **XANES**: 화학종 + 분포 (같은 빔라인: 8-BM-B)
- **광학 현미경**: 형태학적 맥락 + ROI 선택 (ROI-Finder 접근법)

## 출력 형식

| 출력 | 형식 | 내용 |
|------|------|------|
| 원소 맵 | HDF5, TIFF | 원소별 2D 배열 |
| 스펙트럼 | HDF5, ASCII | 에너지-강도 쌍 |
| 정량화 맵 | HDF5, CSV | 농도 값 (µg/cm²) |
| 상관 행렬 | CSV | 원소-원소 상관관계 |
| PCA 결과 | HDF5 | 점수 맵 + 로딩 벡터 |
