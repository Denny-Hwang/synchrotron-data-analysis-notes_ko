# 프로그램 개요

## 미션

**APS BER 프로그램**(아르곤 국립연구소 첨단광자원에 위치한 미국 에너지부 생물환경연구국 프로그램)은
생물학 및 환경과학의 근본적인 문제를 해결하기 위해 특별히 설계된 통합적이고 세계 최고 수준의
X선 역량을 제공합니다.

BER 프로그램은 업그레이드된 APS(APS-U)를 활용하며, 이는 이전보다 최대 500배 밝은 X선 빔을
제공하여 복잡한 생물학적·환경 시스템을 연구하기 위한 전례 없는 공간적·시간적 해상도를
가능하게 합니다.

## 연혁 및 타임라인

| 일자 | 이정표 |
|------|--------|

| **2024년 2분기** | APS-U 업그레이드 건설 완료; 8억 1,500만 달러 투자 |
| **2024년 3분기** | 4세대 MBA 격자로 APS 운영 재개 |
| **2024년 4분기** | BER 프로그램 빔라인 커미셔닝 및 초기 사용자 운영 시작 |
| **2024년 11월** | BER 프로그램 101 워크숍 — 신규 사용자 소개 |

| **2025년 2분기** | 15개 전체 BER 프로그램 빔라인에서 정식 사용자 운영 |

## 연구 분야

BER 프로그램은 7개의 상호 연결된 연구 분야를 지원합니다:

1. **토양과학** — 토양 유기물 순환, 광물-유기물 상호작용, 근권 프로세스
2. **식물과학** — 뿌리 구조, 양분 흡수, 식물-미생물 상호작용
3. **환경화학** — 오염물질 화학종, 생지화학적 순환, 중금속 이동
4. **미생물학** — 미생물 군집 구조, 단일세포 분석, 바이오필름 형성
5. **구조생물학** — 단백질 결정학, 효소 메커니즘, 거대분자 복합체
6. **기후과학** — 대기 에어로졸 특성화, 빙하코어 분석, 탄소 격리
7. **지구화학** — 광물 변환, 유체-암석 상호작용, 핵폐기물 형태

## FICUS 프로그램 통합

**FICUS**(사용자 과학을 위한 시설 통합 협력) 프로그램은 연구자들이 단일 제안서를 통해
여러 DOE 사용자 시설에 접근할 수 있게 합니다. BER 프로그램은 APS X선 역량을 다음과
결합하는 FICUS 파트너십에 참여합니다:

- **EMSL**(환경분자과학연구소) — 질량분석법, NMR, 현미경
- **JGI**(합동유전체연구소) — 유전체학, 메타유전체학, 전사체학
- **NEON**(국립생태관측네트워크) — 생태학 현장 데이터, 원격 탐사

이 통합을 통해 연구자들은 원자 구조(APS)에서 분자 조성(EMSL), 유전체 기능(JGI),
생태계 맥락(NEON)에 이르기까지 동일한 샘플을 여러 길이 스케일과 분석 모달리티로
연구할 수 있습니다.

## 프로그램 조직

BER 프로그램은 다음을 중심으로 구성됩니다:

- 결정학, 이미징, 현미경, 분광학, 산란에 걸친 **15개 전용 빔라인**
- 보완적 분석 역량을 제공하는 **7개 파트너 시설**
- 시설 간 실험 데이터를 연결하는 **통합 데이터 관리**
- 시료 준비, 데이터 분석, 교육을 포함하는 **사용자 지원 인프라**
