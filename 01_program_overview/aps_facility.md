# APS 시설 개요 및 APS-U 업그레이드

## 첨단광자원 (APS)

아르곤 국립연구소의 **첨단광자원(Advanced Photon Source, APS)**은 세계에서 가장 생산성이 높은
싱크로트론 X선 광원 중 하나입니다. 일리노이주 레몬트에 위치한 APS는 매년 다양한 과학 분야에
걸쳐 수천 명의 연구자에게 서비스를 제공합니다.

### 시설 사양

| 파라미터 | 업그레이드 전 (3세대) | 업그레이드 후 APS-U (4세대) |
|----------|----------------------|---------------------------|
| **링 에너지** | 7.0 GeV | 6.0 GeV |
| **둘레** | 1,104 m | 1,104 m |
| **격자 유형** | 이중편향 비색수차 (DBA) | 다중편향 비색수차 (MBA) |
| **에미턴스** | 3.1 nm·rad | 42 pm·rad |
| **밝기 증가** | 기준선 | 최대 500배 |
| **입사** | 탑업(Top-up) | 다중번치 스왑아웃 입사 |
| **빔라인** | 68개 | 68개 이상 (향상된 역량) |
| **전류** | 100 mA | 200 mA (설계) |

## APS-U: 업그레이드

### 개요

**APS 업그레이드(APS-U)**는 기존 저장링을 최첨단 **다중편향 비색수차(MBA) 격자**로 교체한
**8억 1,500만 달러** 규모의 프로젝트입니다. 이 업그레이드로 APS는 3세대에서
**4세대 싱크로트론**으로 전환되어, 원래 광원보다 최대 **500배 밝은** 결맞음 X선 빔을
제공합니다.

### 핵심 기술 개선사항

#### 1. 다중편향 비색수차(MBA) 격자
- 기존 이중편향 비색수차 설계를 대체
- 수평 에미턴스를 줄이기 위해 섹터당 7개의 편향자기(기존 2개 대비) 사용
- **42 pm·rad 에미턴스** 달성 — 회절 한계에 근접
- 결맞음 플럭스가 획기적으로 증가

#### 2. 다중번치 스왑아웃 입사
- 기존 탑업 입사를 대체하는 새로운 입사 방식
- 개별 번치를 "탑업"하는 대신 전체 번치 트레인을 교체
- 보다 안정적이고 균일한 빔 전달 가능
- 유연한 번치 패턴으로 타이밍 모드 실험 지원

#### 3. 향상된 결맞음성
- 결맞음 분율이 경(硬) X선 에너지에서 ~0.1%에서 ~50%로 증가
- 결맞음 이미징 기법(타이코그래피, CDI)이 일상적 방법으로 가능
- 위상대비 이미징 역량이 획기적으로 향상

### 업그레이드 타임라인

| 일자 | 이벤트 |
|------|--------|
| 2019년 | APS-U 프로젝트 CD-3 승인(건설 시작) |
| 2023년 4월 | 설치를 위해 APS 사용자 운영 중단 |
| 2023~2024년 | 저장링 교체 및 설치 |
| 2024년 2분기 | 새 링에서 최초 저장빔 |
| 2024년 3분기 | 빔 커미셔닝 및 빔라인 첫 번째 빛 |
| 2024년 4분기 | 초기 사용자 운영 재개 |
| 2025년 | 향상된 역량으로 정식 사용자 운영 |

## 업그레이드 후 데이터 과제

APS-U는 획기적으로 증가한 데이터 속도로 인해 전례 없는 데이터 과제를 야기합니다:

### 데이터 볼륨 증가

| 모달리티 | 업그레이드 전 | 업그레이드 후 | 배율 |
|----------|-------------|-------------|------|
| **토모그래피** | 10-50 GB/스캔 | 100-500 GB/스캔 | 10배 |
| **타이코그래피** | 5-50 GB/스캔 | 50-500 GB/스캔 | 10배 |
| **XRF 현미경** | 1-10 GB/스캔 | 10-100 GB/스캔 | 10배 |
| **결정학** | 10-100 GB/데이터셋 | 100 GB-1 TB/데이터셋 | 10배 |
| **일일 시설 총량** | ~10 TB/일 | ~100+ TB/일 | 10배 |

### 주요 데이터 과제

1. **실시간 처리**: 기존 재구성 방법이 처리할 수 있는 속도보다 빠르게 데이터가 도착
2. **저장**: 연간 데이터 볼륨이 100 PB를 초과할 것으로 예상
3. **전송**: 컴퓨팅 시설(ALCF)로의 고처리량 데이터 전송 필요
4. **AI/ML 필수**: 이러한 데이터 속도에서는 수동 분석이 불가능
5. **스트리밍 분석**: 사후 분석에서 실시간 스트리밍 분석 패러다임으로 전환

### 컴퓨팅 인프라

- **현장**: 즉시 처리를 위한 APS 컴퓨팅 클러스터
- **ALCF**(아르곤 리더십 컴퓨팅 시설): Aurora 엑사스케일 슈퍼컴퓨터
  - APS와 동일한 ANL 캠퍼스에 위치
  - APS와의 저지연 연결
  - 실험 중 실시간 AI/ML 추론 가능
- **Globus**: APS를 원격 컴퓨팅 자원에 연결하는 고속 데이터 전송 서비스
- **엣지 컴퓨팅**: 실시간 분석을 위한 빔라인 FPGA 및 GPU 기반 엣지 장치

## APS BER 프로그램에 대한 의의

APS-U는 다음을 통해 BER 프로그램 과학에 직접적으로 기여합니다:

- **더 높은 공간 해상도**: 생물학적/환경 시료의 더 미세한 구조 분해
- **더 빠른 데이터 수집**: 동적 과정의 시간분해 연구 가능
- **선량 감소**: 더 낮은 방사선 선량으로 동일한 신호 품질 달성 (생물학적 시료에 중요)
- **결맞음 이미징**: 나노스케일 생물학적 구조를 위한 일상적 타이코그래피 이미징
- **다중모달 역량**: 여러 X선 기법을 동시에 결합할 수 있는 충분한 플럭스
